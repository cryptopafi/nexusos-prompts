# DPM-044 — Diagnose Usability Issues (Norman's 7 Stages Model)

## Problema
Usability problems are identified superficially without understanding where in the cognitive process users fail.

## Procedura
### Prerequisites
- Product with usability issues
- Norman's 7 Stages of Action model knowledge
- User observation data

### Steps
1. Learn Norman's 7 Stages: Goal, Plan, Specify, Perform, Perceive, Interpret, Compare
2. Map the Gulf of Execution: can users figure out what to do? (Plan, Specify, Perform)
3. Map the Gulf of Evaluation: can users tell what happened? (Perceive, Interpret, Compare)
4. Observe 3-5 users attempting a key task and note where they struggle
5. For each struggle, identify which stage fails (e.g., user can't find the button = Specify failure)
6. Classify issues: visibility, feedback, mapping, constraints, affordances, consistency
7. Prioritize fixes by frequency and severity of the stage failure

### Verification
- 7 Stages mapped to product
- User observations completed
- Issues classified and prioritized
