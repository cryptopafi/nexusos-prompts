# DPM-077 — Create Agile-Friendly Team Environment

## Problema
The physical or virtual environment undermines agile practices by making collaboration difficult.

## Procedura
### Prerequisites
- Team workspace (physical or virtual)
- Collaboration tools
- Management support

### Steps
1. Audit the current environment: can the team easily see work, collaborate, and communicate?
2. Physical: create a team area with visible board, whiteboard space, and room for standups
3. Virtual: set up always-on collaboration tools: Slack channels, Miro boards, shared dashboards
4. Ensure information radiators are visible: sprint board, burndown, team charter, metrics
5. Remove collaboration blockers: long approval chains, siloed tools, restricted access
6. Create rituals that build team cohesion: daily standups, weekly demos, retrospectives
7. Gather team feedback monthly on the environment and iterate

### Verification
- Environment audit completed
- Information radiators visible
- Monthly feedback loop established
