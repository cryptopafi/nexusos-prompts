# DPM-057 — Design User Habit Experiments (Hook Framework + Analytics)

## Problema
Products lose users after first use because habitual engagement is not systematically designed and measured.

## Procedura
### Prerequisites
- Hook Model design (DPM-011)
- Analytics tool
- Cohort data

### Steps
1. Define the target habit: what action should users repeat, and how often?
2. Measure current habit metrics: DAU/MAU ratio, retention curve, frequency distribution
3. Identify the weakest Hook component: trigger, action, reward, or investment
4. Design an experiment to strengthen the weakest component
5. Set success criteria: 'Habit is forming if [X%] of users repeat [action] within [Y days]'
6. Run the experiment for 2-4 weeks with a control group
7. Analyze habit formation using retention curves and frequency analysis

### Verification
- Habit metrics baselined
- Experiment designed and run
- Retention curve comparison documented
