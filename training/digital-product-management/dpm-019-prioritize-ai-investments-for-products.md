# DPM-019 — Prioritize AI Investments for Products

## Problema
Teams invest in AI features without a framework for evaluating which AI applications deliver the most product value.

## Procedura
### Prerequisites
- Product roadmap
- AI capability assessment
- User research data

### Steps
1. List all potential AI applications for your product: personalization, automation, prediction, generation, analysis
2. For each, score on: user value (1-5), technical feasibility (1-5), data readiness (1-5), competitive advantage (1-5)
3. Calculate a priority score: (user value x 2 + competitive advantage) x min(feasibility, data readiness)
4. Rank AI investments by priority score
5. For the top 3, define MVP scope and resource requirements
6. Assess build vs. buy for each: use existing AI APIs or build custom models?
7. Create an AI investment roadmap with phased delivery

### Verification
- AI opportunities scored and ranked
- Top 3 MVPs scoped
- AI roadmap created
