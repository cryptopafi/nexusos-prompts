# DPM-074 — Use Trello for Agile Project Management

## Problema
Teams lack a simple, visual tool to manage their agile workflow.

## Procedura
### Prerequisites
- Trello account
- Team members added
- Workflow defined

### Steps
1. Create a Trello board with columns matching your workflow: Backlog, Sprint, In Progress, Review, Done
2. Create cards for each user story with: title, description, acceptance criteria, labels (priority, type)
3. Add members to cards to show ownership
4. Use labels for priority (Critical, High, Medium, Low) and type (Feature, Bug, Tech Debt)
5. Set up a Power-Up for estimation (Planning Poker) or calendar view
6. Run daily standups using the board: each member talks through their cards
7. Archive completed cards at sprint end and create a sprint summary

### Verification
- Trello board configured
- Cards created with full detail
- Board used in daily standups for 1+ sprint
