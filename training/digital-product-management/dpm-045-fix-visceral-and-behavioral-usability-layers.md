# DPM-045 — Fix Visceral and Behavioral Usability Layers

## Problema
Teams fix surface-level UI issues without addressing the deeper visceral and behavioral design layers.

## Procedura
### Prerequisites
- Usability diagnosis (DPM-044)
- Design tool access
- Norman's design level model

### Steps
1. Understand the three levels: Visceral (immediate reaction), Behavioral (usability), Reflective (meaning)
2. Visceral audit: is the first impression positive? Check visual design, layout, color, typography
3. Fix visceral issues: improve visual hierarchy, add white space, align with brand expectations
4. Behavioral audit: can users accomplish their tasks efficiently? Check flows, interactions, error prevention
5. Fix behavioral issues: simplify flows, add inline help, improve error messages, reduce clicks
6. Test fixes with 3 users: did visceral reaction improve? Did task completion improve?
7. Document before/after metrics for each fix

### Verification
- Visceral and behavioral audits completed
- Fixes implemented
- Before/after metrics documented
