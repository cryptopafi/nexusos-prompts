# DPM-061 — Analyze Product Data with Google Analytics

## Problema
Google Analytics is installed but no one uses it for product decisions because reports aren't configured.

## Procedura
### Prerequisites
- Google Analytics 4 (GA4) configured
- Product live with traffic
- Basic analytics knowledge

### Steps
1. Set up key events in GA4: signups, feature usage, purchases, errors
2. Create custom audiences for your user segments (new vs. returning, free vs. paid)
3. Build an acquisition report: which channels bring the most valuable users?
4. Build a behavior report: what do users do after landing? Where do they drop off?
5. Build a retention report: how many users return after day 1, 7, 30?
6. Set up custom explorations for ad-hoc analysis (funnel exploration, path exploration)
7. Create a weekly GA4 review routine and share insights with the product team

### Verification
- Key events configured
- 3 custom reports built
- Weekly review routine established
