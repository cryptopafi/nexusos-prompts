# DPM-022 — Set Up Actionable Product Analytics

## Problema
Analytics are set up to track vanity metrics instead of actionable metrics that drive product decisions.

## Procedura
### Prerequisites
- Analytics tool (Mixpanel, Amplitude, Google Analytics)
- Product live or in beta

### Steps
1. Define your product's key actions: the behaviors that indicate value delivery
2. Map the user funnel: acquisition → activation → engagement → retention → revenue → referral
3. For each funnel stage, define one key metric (e.g., activation = completed onboarding)
4. Set up event tracking for each key action in your analytics tool
5. Create a dashboard with funnel conversion rates and trends
6. Set up cohort analysis to track retention by signup week
7. Establish a weekly analytics review ritual: what changed, why, what to do about it

### Verification
- Key actions and funnel metrics defined
- Event tracking implemented
- Dashboard and weekly review established
