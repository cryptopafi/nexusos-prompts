# PP-012 — Set SMART Goals Using WOOP Method

## Problema
Goal-setting without mental contrasting and implementation intentions fails at a much higher rate than WOOP-enhanced goal-setting.

## Procedura
### Prerequisites
- Basic goal-setting knowledge

### Steps
1. Identify a meaningful goal aligned with personal values
2. W - Wish: articulate the goal clearly and specifically (SMART criteria)
3. O - Outcome: vividly imagine the best possible outcome of achieving the goal
4. O - Obstacle: identify the main internal obstacle that could prevent achievement
5. P - Plan: create if-then implementation intention (if [obstacle], then [action])
6. Write down the complete WOOP for commitment and reference
7. Review and update WOOP weekly, adjusting obstacles and plans as needed

### Verification
- WOOP completed with all four elements for at least one meaningful goal
- Implementation intention is specific and actionable
