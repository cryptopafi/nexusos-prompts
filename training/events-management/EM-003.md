# EM-003 — Build an Event Logistics Checklist

## Problema
Missed logistics details (power, signage, parking, load-in) cause preventable problems that damage the event experience and the brand.

## Procedura
### Prerequisites
- Venue confirmed (EM-002)
- Event plan (EM-001)

### Steps
1. Create the pre-event checklist: venue setup, signage, AV test, catering confirmation, registration setup, emergency plan.
2. Create the day-of checklist: team briefing, vendor check-in, tech run-through, registration open, session management, break management.
3. Create the post-event checklist: vendor settlement, venue walkthrough, equipment return, feedback survey launch, debrief meeting.
4. Assign an owner to every checklist item.
5. Build a timeline that maps each item to a specific date and time.
6. Create a contingency section: what happens if the AV fails, a speaker cancels, or weather disrupts?

### Verification
- All three checklists completed (pre, day-of, post)
- Every item has an owner
- Contingency plans documented for top 3 risks
