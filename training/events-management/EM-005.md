# EM-005 — Create an Event Marketing Strategy

## Problema
Events with poor marketing fail to reach attendance targets regardless of how good the event itself is.

## Procedura
### Prerequisites
- Event plan (EM-001)
- Target audience profile
- Marketing budget

### Steps
1. Define the marketing objective: target registrations/attendance number with deadline.
2. Create the event value proposition: why should someone spend their time attending?
3. Build a marketing timeline: save-the-date (8 weeks out), early bird (6 weeks), full campaign (4 weeks), last call (1 week).
4. Select marketing channels: email, social media, paid ads, partner promotion, PR.
5. Create key assets: event landing page, email sequence (4-6 emails), social media content calendar, and promotional graphics.
6. Set up tracking: UTM parameters, registration source tracking, and conversion funnel metrics.
7. Execute the campaign and optimize weekly based on registration velocity.

### Verification
- Registration target set with deadline
- Marketing timeline covers all 4 phases
- Tracking implemented for all channels
