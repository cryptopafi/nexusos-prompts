# EM-004 — Coordinate and Manage an Event Team

## Problema
Event teams without clear roles and communication protocols create duplicated effort, missed tasks, and stressful event days.

## Procedura
### Prerequisites
- Event plan (EM-001)
- Team member list

### Steps
1. Define all event roles: event lead, registration, AV/tech, catering liaison, speaker coordinator, volunteer manager.
2. Create a RACI matrix: who is Responsible, Accountable, Consulted, and Informed for each event workstream.
3. Set up a communication channel (Slack, WhatsApp group) with naming convention for urgent vs. routine messages.
4. Schedule check-in meetings: weekly during planning, daily in the final week, hourly on event day.
5. Create a contact sheet with every team member, vendor, and venue contact with phone numbers.
6. Run a team briefing 24 hours before the event covering roles, timeline, and escalation protocol.

### Verification
- RACI matrix completed
- Communication channel set up and tested
- Team briefing conducted with all members
