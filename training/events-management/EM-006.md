# EM-006 — Build an Event Budget Template

## Problema
Events consistently go over budget because planners underestimate costs, forget categories, and lack a tracking mechanism.

## Procedura
### Prerequisites
- Event plan (EM-001)
- Venue quotes (EM-002)

### Steps
1. Create budget categories: venue, catering, AV/technology, marketing, speakers/entertainment, staffing, materials/printing, contingency.
2. For each category, list all line items with estimated costs (get 2-3 quotes where possible).
3. Add a 10-15% contingency line for unexpected costs.
4. Track revenue sources if applicable: ticket sales, sponsorships, exhibitor fees.
5. Calculate break-even point: minimum attendance or sponsorship needed to cover costs.
6. Set up a live tracking spreadsheet with columns: budgeted, actual, variance, and notes.
7. Review budget weekly during planning phase and daily during event week.

### Verification
- All budget categories populated with estimates
- Contingency included at 10-15%
- Break-even point calculated
