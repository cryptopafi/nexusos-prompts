# EM-001 — Plan an Event Using the 5-Stage Planning Process

## Problema
Events fail when planners jump to execution without working through a structured planning process that covers all critical dimensions.

## Procedura
### Prerequisites
- Event objective defined
- Stakeholder requirements gathered

### Steps
1. Stage 1 - Research: Define the event purpose, target audience, and success criteria.
2. Stage 2 - Design: Choose format (in-person, virtual, hybrid), theme, and key experiences.
3. Stage 3 - Planning: Create the master timeline with milestones for venue, vendors, marketing, and logistics.
4. Stage 4 - Coordination: Assign roles, confirm all vendors, run a pre-event walkthrough.
5. Stage 5 - Evaluation: Define post-event metrics (attendance, satisfaction, ROI) and feedback collection method.
6. Document the complete plan in a single event brief that all stakeholders can reference.

### Verification
- All 5 stages documented
- Master timeline created with milestones
- Event brief shared with all stakeholders
