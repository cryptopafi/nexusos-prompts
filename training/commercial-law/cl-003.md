# CL-003 — Determine Valid Acceptance of a Contract Offer

## Problema
Invalid acceptance (counter-offers, conditional acceptance, late acceptance) can void expected contract formation.

## Procedura
### Prerequisites
- CL-002 completed

### Steps
1. Verify the acceptance is by an authorized offeree
2. Check that acceptance mirrors the offer terms exactly (mirror image rule for common law)
3. Apply UCC 2-207 for additional/different terms in sale of goods
4. Determine if acceptance method complies with any stipulated medium
5. Apply the mailbox rule for timing of acceptance (effective upon dispatch)
6. Identify if silence can constitute acceptance (prior dealings, unsolicited goods)

### Verification
- Mirror image rule or UCC 2-207 correctly applied based on transaction type
- Timing of acceptance determined using correct rule
