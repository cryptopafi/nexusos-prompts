# PBR-022 — Set Up Facebook Pixel and Custom Audiences

## Problema
Without a pixel and custom audiences, your ad spend is blind: you cannot track conversions, retarget visitors, or build lookalike audiences.

## Procedura
### Prerequisites
- Business Manager set up (PBR-020)
- Website access for pixel installation

### Steps
1. Navigate to Events Manager in Business Manager and create a new Pixel.
2. Install the Pixel base code on every page of your website (use Google Tag Manager for easy management).
3. Set up standard events: PageView (automatic), ViewContent, AddToCart, InitiateCheckout, Purchase.
4. Verify pixel firing using the Facebook Pixel Helper Chrome extension.
5. Create Custom Audiences: All website visitors (180 days), specific page visitors, time-on-site top 25%.
6. Create Lookalike Audiences: 1% of each Custom Audience in your target country.
7. Test pixel data: visit your own site and check Events Manager for recorded events.

### Verification
- Pixel installed and verified with Pixel Helper
- Standard events firing correctly
- At least 3 Custom Audiences and 3 Lookalike Audiences created
