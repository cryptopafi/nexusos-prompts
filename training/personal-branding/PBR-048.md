# PBR-048 — Create Amazon Low-Content KDP Books

## Problema
Low-content books (journals, planners, workbooks) on Amazon KDP generate passive income with minimal writing and can reinforce your personal brand.

## Procedura
### Prerequisites
- Amazon KDP account
- Canva or design tool

### Steps
1. Set up an Amazon KDP account at kdp.amazon.com (free).
2. Research profitable low-content niches: use Amazon search, Book Bolt, or KDP Spy to find categories with demand and manageable competition.
3. Create your interior: use Canva or Book Bolt templates for journals (lined, dotted, grid), planners, trackers, or workbooks.
4. Design a cover that stands out: bold title, clean design, genre-appropriate style. Cover sells the book.
5. Write your title and subtitle using Amazon SEO: include keywords your target buyer would search.
6. Write a compelling book description: benefit-focused, with formatting (bold, bullets) for readability.
7. Publish and order a proof copy to verify print quality before promoting.

### Verification
- KDP account set up
- First book published with optimized title and cover
- Proof copy reviewed for quality
