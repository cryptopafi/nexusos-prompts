# PBR-033 — Create YouTube Thumbnails with Canva

## Problema
Thumbnails drive 50%+ of click-through decisions. A consistent, high-contrast thumbnail style builds channel recognition.

## Procedura
### Prerequisites
- Canva account (free)
- Brand visual identity defined

### Steps
1. Create a thumbnail template in Canva at 1280x720 pixels (16:9 ratio).
2. Include 3 elements maximum: face/image (left third), text overlay (right two-thirds), background.
3. Use large, bold text (4-6 words max). Text must be readable at mobile thumbnail size.
4. Apply high-contrast colors: your text and image must pop against the background.
5. Include a face with emotion if possible: surprise, excitement, confusion. Faces drive clicks.
6. Create 3 thumbnail variations for each video and choose the one that stands out most in a row of competitors.
7. Maintain a consistent thumbnail style: same fonts, color palette, and layout structure across all videos.

### Verification
- Thumbnail template created at correct dimensions
- 3 variations created per video
- Consistent style maintained across 10+ videos
