# PBR-005 — Create a Personal Brand Name and Domain

## Problema
Using an inconsistent name across platforms confuses your audience and weakens search discoverability.

## Procedura
### Prerequisites
- Niche defined (PBR-001)

### Steps
1. Decide between using your real name or a brand name. Real name builds authority; brand name is more sellable.
2. If using a brand name, generate 20 options using: niche keywords, personality traits, and audience aspirations.
3. Check availability across all platforms: domain (.com), Instagram, YouTube, TikTok, Twitter/X, LinkedIn.
4. Use Namechk.com or similar tool to check cross-platform availability simultaneously.
5. Secure the domain name immediately (even before finalizing: domains are cheap, lost names are expensive).
6. Register the name on all major social platforms, even those you won't use immediately.

### Verification
- Name chosen with clear rationale
- Domain secured
- Name registered on at least 5 major platforms
