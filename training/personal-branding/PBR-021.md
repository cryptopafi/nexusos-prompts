# PBR-021 — Create an Instagram Ad Campaign (Full Funnel)

## Problema
Running only bottom-funnel ads wastes budget on cold audiences. A full-funnel campaign warms audiences before asking for the conversion.

## Procedura
### Prerequisites
- Business Manager set up (PBR-020)
- Pixel and audiences configured (PBR-022)

### Steps
1. Create Top of Funnel (TOF) campaign: objective = Reach or Video Views. Target Lookalike Audiences. Content = educational Reel or carousel.
2. Create Middle of Funnel (MOF) campaign: objective = Engagement or Traffic. Retarget TOF viewers (video watchers, post engagers). Content = case study or testimonial.
3. Create Bottom of Funnel (BOF) campaign: objective = Conversions. Retarget MOF engagers + website visitors. Content = direct offer with urgency.
4. Set budget allocation: 60% TOF, 25% MOF, 15% BOF.
5. Create 3 ad variations per funnel stage for testing (different hooks, visuals, or copy).
6. Launch campaigns and monitor daily for the first week: check CPM, CTR, and CPC by funnel stage.
7. Optimize after 7 days: pause underperforming ads, increase budget on winners, refresh creative monthly.

### Verification
- 3 funnel stages active with correct objectives
- Budget allocated across funnel stages
- 3 ad variations per stage for testing
