# PBR-026 — Create and Run TikTok Ads

## Problema
TikTok ads that look like ads get skipped. Native-looking content ads outperform polished creative on TikTok.

## Procedura
### Prerequisites
- TikTok Business Center account
- Ad budget

### Steps
1. Set up TikTok Business Center and create an ad account at ads.tiktok.com.
2. Install the TikTok Pixel on your website for conversion tracking.
3. Choose your campaign objective: Traffic (awareness), Conversions (sales), or Lead Generation.
4. Create your ad group: target by interest, behavior, and lookalike audiences. Start broad and let TikTok optimize.
5. Create ad creative that looks native: filmed vertically, uses trending sounds, feels like organic content, not a commercial.
6. Set budget: start with $20-50/day for testing. Run for 7 days before judging performance.
7. Optimize after 7 days: double down on winning creative, test new hooks, and narrow audiences based on data.

### Verification
- TikTok Pixel installed and verified
- First campaign launched with native-looking creative
- 7-day test completed with performance data analyzed
