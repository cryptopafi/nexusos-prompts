# SE-038 — Create a Sales Pitch Presentation Using AI Copilot

## Problema
Sales presentations take hours to build and often miss the mark on structure and messaging.

## Procedura
### Prerequisites
- AI presentation tool (Gamma, Beautiful.ai, or Copilot in PowerPoint)
- Discovery notes from prospect

### Steps
1. Gather inputs: prospect pain points, your solution's value props, relevant case studies, pricing options
2. Prompt AI: 'Create a 10-slide sales presentation for [prospect] in [industry] addressing [pain points]'
3. Review AI-generated structure and adjust flow: Problem → Impact → Solution → Proof → Pricing → Next Steps
4. Replace generic AI content with specific prospect data and real case studies
5. Add your branding, visuals, and speaker notes to each slide
6. Practice the presentation with a colleague and get feedback
7. Save the template for reuse and document the AI prompt that worked best

### Verification
- AI-generated presentation created
- Customized with real prospect data
- Practice run completed
