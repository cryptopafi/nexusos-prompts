# SE-036 — Measure AI Sales Performance Metrics

## Problema
Teams adopt AI tools but never measure whether they actually improve sales outcomes.

## Procedura
### Prerequisites
- AI tools in use
- CRM data
- Pre-AI baseline metrics

### Steps
1. Document baseline metrics before AI adoption: conversion rate, deal velocity, activity volume, win rate
2. Define AI-specific metrics: AI-assisted deal win rate, time saved per rep, forecast accuracy improvement
3. Set up a comparison dashboard: AI-assisted vs. non-AI-assisted deals
4. Track AI tool adoption: which reps use AI tools and how frequently
5. Measure ROI: revenue impact of AI tools vs. cost of tools
6. Run monthly AI performance reviews with the team
7. Create an AI sales scorecard combining traditional and AI-specific metrics

### Verification
- Baseline metrics documented
- AI performance dashboard live
- Monthly review cadence started
