# SE-028 — Implement Data-Backed Personalization in Sales

## Problema
Sales outreach is one-size-fits-all, ignoring available data that could dramatically improve relevance.

## Procedura
### Prerequisites
- CRM data
- Marketing automation data
- AI personalization tool

### Steps
1. Audit available data: website visits, content downloads, email engagement, past purchases
2. Define personalization triggers: 'If prospect visited pricing page, send case study email'
3. Create 5 personalization rules mapping data signals to sales actions
4. Use AI to generate personalized content snippets based on prospect data
5. Implement personalization in your outreach tool (email sequences, call scripts)
6. Test personalized vs. generic outreach on a 200-contact split
7. Measure conversion uplift and refine rules based on results

### Verification
- 5 personalization rules defined
- Split test executed
- Conversion uplift measured
