# SE-016 — Qualify Deals to Avoid Wasting Time on Unqualified Leads

## Problema
Pipeline is bloated with unqualified deals that consume rep time but never close.

## Procedura
### Prerequisites
- Active pipeline with 10+ opportunities
- BANT or MEDDIC framework knowledge

### Steps
1. Choose a qualification framework: BANT (simple) or MEDDIC (complex B2B)
2. For MEDDIC: assess Metrics, Economic Buyer, Decision Criteria, Decision Process, Identify Pain, Champion
3. Review each active deal and score on the chosen framework
4. Flag deals scoring below threshold as 'at risk' in CRM
5. For at-risk deals, define one specific action to improve qualification (e.g., get access to economic buyer)
6. Set a deadline: if qualification doesn't improve in 2 weeks, move to 'closed-lost'
7. Create a pipeline hygiene routine: qualify every new deal within 48 hours of entry

### Verification
- All active deals scored
- At-risk deals flagged with action items
- Pipeline hygiene routine documented
