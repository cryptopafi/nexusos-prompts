# SE-029 — Build and Manage a Sales Pipeline (Step-by-Step)

## Problema
New reps lack a structured pipeline and lose track of deals across different stages.

## Procedura
### Prerequisites
- CRM access
- Understanding of sales stages

### Steps
1. Define your pipeline stages with clear exit criteria for each
2. Add all current opportunities to the pipeline with accurate stage assignments
3. Set expected close dates based on your average sales cycle length
4. Calculate pipeline coverage ratio: pipeline value / quota (target 3x)
5. Identify stage where most deals are stuck and create an action plan
6. Set up a daily pipeline review routine (10 min): update stages, add next actions
7. Create a weekly pipeline report showing movement, additions, and losses

### Verification
- Pipeline built with all deals staged
- Coverage ratio calculated
- Weekly report generated
