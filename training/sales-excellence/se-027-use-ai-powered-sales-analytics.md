# SE-027 — Use AI-Powered Sales Analytics

## Problema
Sales leaders rely on lagging indicators and static reports instead of predictive AI analytics.

## Procedura
### Prerequisites
- CRM with analytics
- AI analytics tool (Gong, Clari, InsightSquared)
- Historical data

### Steps
1. Identify 3 questions AI analytics should answer: Which deals will close? Which reps need help? Where is pipeline leaking?
2. Connect AI analytics tool to CRM data source
3. Set up predictive deal scoring: let AI flag deals likely to slip or close
4. Configure conversation intelligence: analyze call recordings for winning patterns
5. Build an AI-powered dashboard with leading indicators (engagement trends, sentiment, activity patterns)
6. Review AI insights weekly and take action on flagged items
7. Track prediction accuracy monthly and recalibrate as needed

### Verification
- AI analytics tool connected
- Predictive dashboard live
- Weekly review cadence established
