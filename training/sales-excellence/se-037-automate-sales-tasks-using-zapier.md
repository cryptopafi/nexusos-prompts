# SE-037 — Automate Sales Tasks Using Zapier

## Problema
Reps waste hours on repetitive tasks (data entry, follow-ups, notifications) that can be automated.

## Procedura
### Prerequisites
- Zapier account
- CRM access
- Email and calendar tools

### Steps
1. List the 5 most repetitive sales tasks: data entry, follow-up reminders, lead routing, reporting, notifications
2. Prioritize the top 3 by time saved and ease of automation
3. Build Zap 1: When new lead enters CRM → send welcome email + create task
4. Build Zap 2: When deal moves to 'Proposal' stage → notify manager + create calendar reminder
5. Build Zap 3: When no activity on deal for 7 days → send rep reminder + update deal tag
6. Test each Zap with sample data and verify correct execution
7. Calculate time saved per week and document the automation playbook

### Verification
- 3 Zaps built and tested
- Time savings calculated
- Automation playbook documented
