# OC-003 — Select Change Management Strategies (Kotter, ADKAR, Lewin)

## Problema
Using the wrong change management model for the situation leads to either oversimplification or analysis paralysis.

## Procedura
### Prerequisites
- Understanding of the change context and organizational culture

### Steps
1. Characterize the change: scope (incremental vs transformational), urgency, complexity
2. Evaluate Lewin's 3-Stage Model: suitable for simple, clearly bounded changes
3. Evaluate Kotter's 8-Step Process: suitable for large-scale transformational change
4. Evaluate ADKAR Model: suitable for individual-focused adoption and tracking
5. Assess organizational culture fit with each model's assumptions
6. Select primary model and identify supplementary elements from others
7. Adapt selected model to organizational context (do not apply rigidly)

### Verification
- At least 3 models evaluated against the specific change context
- Selection rationale documented with organizational culture considerations
