# OC-001 — Develop a Change Management Communication Framework

## Problema
Change initiatives fail primarily due to poor communication, not poor strategy, making a communication framework essential.

## Procedura
### Prerequisites
- Understanding of the planned organizational change

### Steps
1. Define the change narrative: why the change is happening, what it means, and how it benefits stakeholders
2. Identify all audience segments and their information needs
3. Establish communication principles: transparency, consistency, two-way dialogue
4. Design message hierarchy: vision > rationale > impact > action > support
5. Select communication channels appropriate to each audience (town halls, email, intranet, managers)
6. Create feedback mechanisms for each audience segment
7. Align executive messaging to ensure consistent voice from leadership

### Verification
- Communication framework covers all identified audience segments
- Message hierarchy documented with specific talking points per level
