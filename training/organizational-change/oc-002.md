# OC-002 — Create a Change Management Communication Plan Checklist

## Problema
Ad hoc communication during change creates information vacuums that employees fill with rumors and resistance.

## Procedura
### Prerequisites
- OC-001 completed

### Steps
1. Map communication milestones to change project timeline
2. Assign message owner and approver for each communication
3. Define delivery channel, format, and timing for each message
4. Create message templates for recurring communications (weekly updates, FAQs)
5. Establish escalation protocol for misinformation or rumors
6. Schedule manager cascade briefings ahead of broad communications
7. Plan communication effectiveness measurement (surveys, feedback, engagement metrics)
8. Build communication calendar with all touchpoints visible

### Verification
- Communication calendar covers entire change timeline
- Every communication has assigned owner, channel, timing, and approval
