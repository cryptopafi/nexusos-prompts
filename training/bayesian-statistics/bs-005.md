# BS-005 — Calculate Posterior Credible Intervals

## Problema
Point estimates alone hide uncertainty; credible intervals quantify what the data supports.

## Procedura
### Prerequisites
- R or Python
- posterior distribution (BS-004)

### Steps
1. Compute equal-tailed 95% credible interval: find 2.5th and 97.5th percentiles.
2. Use `qbeta()` (R) or `scipy.stats.beta.ppf()` for conjugate posteriors.
3. Compute Highest Posterior Density (HPD) interval using numerical methods.
4. Compare equal-tailed vs HPD interval widths.
5. Interpret: 95% probability the parameter lies in this interval (Bayesian interpretation).
6. Contrast with frequentist confidence interval interpretation.

### Verification
- Credible interval contains the true parameter in simulation.
- HPD interval is narrower than equal-tailed for skewed posteriors.
