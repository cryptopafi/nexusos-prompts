# BS-003 — Plot Likelihood Functions in R

## Problema
Abstract likelihood concepts become concrete only through visualization of the likelihood surface.

## Procedura
### Prerequisites
- R installed
- ggplot2 package

### Steps
1. Define a grid of parameter values: `theta <- seq(0, 1, length=100)`.
2. Compute likelihood at each grid point for your data.
3. Plot with `ggplot`: x=theta, y=likelihood.
4. Add vertical line at MLE: `geom_vline(xintercept=mle)`.
5. Normalize to get relative likelihood and overlay on same plot.
6. Create log-likelihood plot for better visualization of tails.

### Verification
- Likelihood plot shows clear peak at MLE.
- Log-likelihood plot reveals curvature for inference.
