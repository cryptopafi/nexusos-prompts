# BS-013 — Perform Bayesian Linear Regression in R

## Problema
Standard linear regression provides only point estimates; Bayesian regression quantifies full parameter uncertainty.

## Procedura
### Prerequisites
- R installed
- dataset with continuous response

### Steps
1. Specify model: Y = X*beta + epsilon, epsilon ~ N(0, sigma^2).
2. Choose priors: Normal for beta coefficients, Inverse-Gamma for sigma^2.
3. Derive posterior: Normal-Inverse-Gamma conjugate update.
4. Compute posterior means and credible intervals for all beta coefficients.
5. Generate posterior predictive distribution for new observations.
6. Plot credible bands around the regression line.
7. Compare with `lm()` frequentist estimates.

### Verification
- Posterior credible intervals cover frequentist point estimates.
- Predictive intervals wider than credible intervals (as expected).
