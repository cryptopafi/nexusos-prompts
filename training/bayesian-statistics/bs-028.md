# BS-028 — Fit Gaussian Mixture Model Using EM Algorithm

## Problema
EM provides fast point estimation for mixture models before committing to full Bayesian MCMC.

## Procedura
### Prerequisites
- R or Python

### Steps
1. Initialize K component means, variances, and mixing weights.
2. E-step: compute responsibility of each component for each observation.
3. M-step: update means, variances, and weights using responsibilities.
4. Iterate E-M steps until log-likelihood converges (change < threshold).
5. Monitor log-likelihood: must be non-decreasing.
6. Run with multiple random initializations to avoid local optima.
7. Plot fitted mixture density overlaid on data histogram.

### Verification
- Log-likelihood converges monotonically.
- Fitted mixture matches data distribution visually.
