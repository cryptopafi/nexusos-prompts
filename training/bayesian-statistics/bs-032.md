# BS-032 — Apply Mixture Models for Classification

## Problema
Discriminative classifiers ignore data generation process; mixture-based classification models class densities.

## Procedura
### Prerequisites
- R or Python
- BS-031 completed

### Steps
1. Fit separate mixture models for each class: `f_k(x)` for class k.
2. Compute class priors from training data: `pi_k = n_k / n`.
3. Classify new observation using Bayes rule: `P(class=k|x) = pi_k * f_k(x) / f(x)`.
4. Compute posterior class probabilities for test set.
5. Evaluate: accuracy, confusion matrix, ROC curve.
6. Compare with logistic regression and naive Bayes classifiers.

### Verification
- Classification accuracy competitive with standard methods.
- Posterior class probabilities well-calibrated.
