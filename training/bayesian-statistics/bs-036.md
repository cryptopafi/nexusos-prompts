# BS-036 — Perform Bayesian Inference in AR(p) Models

## Problema
Higher-order AR models require careful parameterization to maintain stationarity in Bayesian fitting.

## Procedura
### Prerequisites
- R
- JAGS
- BS-035 completed

### Steps
1. Extend AR(1) to AR(p): `y[t] ~ dnorm(mu + sum(phi[k]*(y[t-k] - mu)), tau)` for k=1..p.
2. Specify priors: phi[k] ~ dnorm(0, 1) with stationarity checked post-hoc.
3. Prepare data: time series, order p, and first p values as initial conditions.
4. Run MCMC and check convergence for all phi coefficients.
5. Post-process: check that posterior samples yield stationary polynomials.
6. Compare models AR(1), AR(2), AR(3) using DIC.
7. Report posterior summaries for all coefficients with credible intervals.

### Verification
- Model converges for all AR coefficients.
- DIC comparison identifies preferred model order.
