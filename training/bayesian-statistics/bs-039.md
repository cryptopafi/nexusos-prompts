# BS-039 — Perform Smoothing and Forecasting with NDLM

## Problema
Filtering uses only past data; smoothing uses the full dataset for better historical estimates.

## Procedura
### Prerequisites
- R
- dlm package
- BS-038 completed

### Steps
1. Run Kalman smoother: `dlmSmooth(filtered_output)`.
2. Compare filtered vs smoothed estimates: smoothed should be less variable.
3. Compute forecast: `dlmForecast(filtered_output, nAhead=h)`.
4. Extract forecast means and variances for h-step ahead predictions.
5. Plot: historical smoothed estimates + forecast with prediction intervals.
6. Assess forecast quality with rolling origin cross-validation.

### Verification
- Smoothed estimates less variable than filtered.
- Forecast prediction intervals cover future observations.
