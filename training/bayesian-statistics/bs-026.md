# BS-026 — Build Bayesian Hierarchical Model in JAGS

## Problema
Hierarchical models share information across groups, producing better estimates than separate or pooled analyses.

## Procedura
### Prerequisites
- R
- JAGS
- BS-014 and BS-020 completed

### Steps
1. Write JAGS model with group-level parameters drawn from population distribution.
2. Example: `mu[j] ~ dnorm(mu0, tau0)` for group means, `mu0 ~ dnorm(0, 0.001)` for population mean.
3. Add observation level: `y[i] ~ dnorm(mu[group[i]], tau_y)`.
4. Prepare data: observations, group indices, number of groups.
5. Run MCMC with monitoring of both group and population parameters.
6. Check convergence at all levels of the hierarchy.
7. Visualize shrinkage: plot group estimates vs raw group means.

### Verification
- Model converges for all parameters.
- Shrinkage plot shows group estimates pulled toward population mean.
