# BS-017 — Implement Gibbs Sampler

## Problema
Models with multiple parameters benefit from Gibbs sampling when full conditionals are tractable.

## Procedura
### Prerequisites
- R or Python
- BS-016 completed

### Steps
1. Derive full conditional distributions for each parameter.
2. Initialize all parameters to starting values.
3. Iterate: sample each parameter from its full conditional, given current values of others.
4. Run for N iterations with burn-in.
5. Assess convergence of each parameter's chain.
6. Compute joint and marginal posterior summaries.

### Verification
- Each parameter's marginal converges to known distribution.
- Joint posterior samples show correct correlation structure.
