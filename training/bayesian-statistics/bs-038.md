# BS-038 — Implement Normal Dynamic Linear Model (NDLM) Filtering

## Problema
Static regression cannot capture time-varying relationships; DLMs allow parameters to evolve over time.

## Procedura
### Prerequisites
- R installed
- dlm package

### Steps
1. Install: `install.packages('dlm')`.
2. Define NDLM: observation equation `Y_t = F_t * theta_t + v_t` and state equation `theta_t = G_t * theta_{t-1} + w_t`.
3. Specify model matrices F, G and variance matrices V, W.
4. Build model in R: `dlm(FF=F, GG=G, V=V, W=W, m0=m0, C0=C0)`.
5. Run Kalman filter: `dlmFilter(y, model)`.
6. Extract filtered state estimates and their variances.
7. Plot filtered estimates with uncertainty bands over time.

### Verification
- Kalman filter runs without errors.
- Filtered states track the true time-varying signal.
