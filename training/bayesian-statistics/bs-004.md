# BS-004 — Derive and Compute Bayesian Posterior Distributions

## Problema
Computing posteriors manually builds intuition that black-box MCMC tools cannot provide.

## Procedura
### Prerequisites
- R or Python
- BS-001 completed

### Steps
1. Choose a prior distribution for the parameter of interest.
2. Write the likelihood function for the observed data.
3. Multiply prior times likelihood to get the unnormalized posterior.
4. Compute the normalizing constant by integration (analytically or numerically).
5. Plot prior, likelihood, and posterior on the same graph.
6. Compute posterior mean, mode, and variance.

### Verification
- Posterior integrates to 1 (properly normalized).
- Posterior lies between prior and likelihood as expected.
