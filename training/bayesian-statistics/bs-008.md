# BS-008 — Fit Conjugate Models for Binomial Data (Beta-Binomial)

## Problema
The Beta-Binomial is the foundational conjugate model for learning proportions from binary data.

## Procedura
### Prerequisites
- R or Python
- BS-004 completed

### Steps
1. Specify Beta(a, b) prior for the success probability p.
2. Observe data: y successes in n trials.
3. Derive posterior: Beta(a + y, b + n - y).
4. Compute posterior mean: (a + y) / (a + b + n).
5. Plot prior Beta(a,b) and posterior Beta(a+y, b+n-y) together.
6. Compute 95% credible interval using `qbeta()`.

### Verification
- Posterior parameters equal prior + data counts.
- Posterior shifts toward data as sample size increases.
