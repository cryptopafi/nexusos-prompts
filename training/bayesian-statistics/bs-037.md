# BS-037 — Select Model Order Using AIC/BIC/DIC

## Problema
Picking model complexity by eye or tradition leads to overfitting or underfitting time series data.

## Procedura
### Prerequisites
- R
- fitted models of different orders

### Steps
1. Compute AIC for each model: `AIC = -2*loglik + 2*p`.
2. Compute BIC for each model: `BIC = -2*loglik + p*log(n)`.
3. Compute DIC from JAGS: `dic.samples(model, n.iter)`.
4. Create comparison table: model, parameters, AIC, BIC, DIC.
5. Select model minimizing each criterion (they may disagree).
6. Discuss trade-offs: BIC more conservative, DIC Bayesian-specific.
7. Report selected model with justification.

### Verification
- Comparison table complete with all criteria.
- Selected model order justified with information criteria.
