# BS-033 — Determine Number of Components Using BIC

## Problema
Overfitting mixture models with too many components produces spurious clusters; BIC penalizes complexity.

## Procedura
### Prerequisites
- R or Python
- BS-028 completed

### Steps
1. Fit mixture models for K = 1, 2, ..., K_max.
2. For each K, compute BIC = -2 * log-likelihood + p * log(n) where p = number of parameters.
3. Count parameters: K means + K variances + (K-1) weights = 3K - 1 (univariate).
4. Plot BIC vs K: select K at the minimum (elbow).
5. Alternatively compute AIC and compare model selection.
6. Validate selected K with posterior predictive checks.

### Verification
- BIC plot shows clear minimum at optimal K.
- Selected K matches true number of components in simulated data.
