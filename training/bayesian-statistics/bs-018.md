# BS-018 — Set Up and Run JAGS Models in R

## Problema
Hand-coded MCMC is error-prone; JAGS provides validated, efficient sampling for standard models.

## Procedura
### Prerequisites
- R installed
- JAGS installed
- rjags R package

### Steps
1. Install JAGS: `brew install jags` (macOS) or `apt install jags` (Linux).
2. Install R package: `install.packages('rjags')`.
3. Write JAGS model string defining likelihood and priors.
4. Prepare data as a named list.
5. Initialize model: `jags.model(textConnection(model_string), data=data_list)`.
6. Run MCMC: `coda.samples(model, variable.names, n.iter)`.
7. Extract and summarize posterior samples with `summary()`.

### Verification
- JAGS model compiles without errors.
- Posterior samples extractable as coda mcmc object.
