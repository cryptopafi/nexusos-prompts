# BS-001 — Apply Bayes Theorem to Update Beliefs from Data

## Problema
Practitioners default to frequentist methods, missing the ability to incorporate prior knowledge into inference.

## Procedura
### Prerequisites
- R or Python installed
- basic probability knowledge

### Steps
1. Write out Bayes' theorem: P(H|D) = P(D|H) * P(H) / P(D).
2. Define a prior probability P(H) for your hypothesis.
3. Compute the likelihood P(D|H) from observed data.
4. Calculate the marginal likelihood P(D) by summing over all hypotheses.
5. Compute the posterior P(H|D) and interpret the result.
6. Repeat with different priors to observe sensitivity.

### Verification
- Posterior probability computed correctly.
- Changing priors shifts posterior in expected direction.
