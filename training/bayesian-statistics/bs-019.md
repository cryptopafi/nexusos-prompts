# BS-019 — Assess MCMC Convergence

## Problema
Using unconverged MCMC output produces invalid posterior inference and misleading conclusions.

## Procedura
### Prerequisites
- R with coda package
- MCMC output from JAGS or custom sampler

### Steps
1. Run multiple chains with different starting values.
2. Plot trace plots: look for stationarity and mixing.
3. Compute Gelman-Rubin R-hat statistic: `gelman.diag()`. Target R-hat < 1.05.
4. Check effective sample size: `effectiveSize()`. Need ESS > 400 for reliable intervals.
5. Plot autocorrelation: `autocorr.plot()`. High autocorrelation means poor mixing.
6. If not converged: increase burn-in, thin chains, or reparameterize model.

### Verification
- R-hat < 1.05 for all parameters.
- Effective sample size > 400.
