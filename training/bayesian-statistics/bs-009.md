# BS-009 — Fit Conjugate Models for Poisson Data (Gamma-Poisson)

## Problema
Count data modeling with Gamma-Poisson conjugacy enables closed-form rate inference.

## Procedura
### Prerequisites
- R or Python
- BS-004 completed

### Steps
1. Specify Gamma(a, b) prior for the Poisson rate lambda.
2. Observe data: counts y_1, ..., y_n.
3. Derive posterior: Gamma(a + sum(y), b + n).
4. Compute posterior mean: (a + sum(y)) / (b + n).
5. Plot prior and posterior Gamma distributions.
6. Compute 95% credible interval using `qgamma()`.

### Verification
- Posterior parameters correctly combine prior and sufficient statistics.
- Credible interval narrows with more data.
