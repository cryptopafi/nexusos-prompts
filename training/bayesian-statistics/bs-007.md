# BS-007 — Perform Prior Predictive Simulation

## Problema
Checking if a prior implies reasonable data outcomes catches misspecified priors before fitting.

## Procedura
### Prerequisites
- R or Python
- model specification

### Steps
1. Sample parameters from the prior distribution.
2. For each sampled parameter, simulate a dataset from the likelihood.
3. Plot histograms of simulated data summaries (means, ranges, extremes).
4. Check if simulated data looks plausible compared to domain knowledge.
5. If prior predictive is unreasonable, adjust prior hyperparameters.
6. Repeat until prior predictive simulation passes sanity checks.

### Verification
- Simulated datasets fall within plausible ranges.
- Adjusted prior produces more reasonable predictions.
