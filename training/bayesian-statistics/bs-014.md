# BS-014 — Specify a Bayesian Hierarchical Model

## Problema
Non-hierarchical models ignore group structure, leading to either over-pooling or under-pooling estimates.

## Procedura
### Prerequisites
- R or Python
- understanding of random effects

### Steps
1. Identify the hierarchical structure: observations within groups.
2. Write level 1: Y_ij | theta_j ~ f(theta_j) for observations within group j.
3. Write level 2: theta_j | phi ~ g(phi) for group-level parameters.
4. Write level 3: phi ~ h() for hyperpriors on population parameters.
5. Draw the directed acyclic graph (DAG) of the model.
6. Identify which parameters are exchangeable.
7. Describe how partial pooling arises from the hierarchical structure.

### Verification
- Model DAG correctly represents conditional dependencies.
- Three levels clearly specified with distributions.
