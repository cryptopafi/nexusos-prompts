# BS-006 — Select and Evaluate Prior Distributions

## Problema
Poor prior selection either dominates the posterior (too informative) or provides no regularization.

## Procedura
### Prerequisites
- R or Python
- understanding of conjugate families

### Steps
1. List common prior families: Beta, Normal, Gamma, Uniform.
2. Match prior to parameter space: Beta for [0,1], Gamma for (0,inf).
3. Set hyperparameters reflecting prior knowledge or desired vagueness.
4. Plot several candidate priors with different hyperparameters.
5. Compute posterior under each prior and compare sensitivity.
6. Document prior choice rationale for reproducibility.

### Verification
- Prior support matches parameter space.
- Posterior is robust across reasonable prior choices.
