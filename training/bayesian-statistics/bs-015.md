# BS-015 — Perform Monte Carlo Integration and Error Estimation

## Problema
Many Bayesian posteriors lack closed-form integrals; Monte Carlo methods provide numerical solutions.

## Procedura
### Prerequisites
- R or Python

### Steps
1. Define the integral to approximate: E[g(theta)] under posterior.
2. Sample N draws from the posterior (or proposal) distribution.
3. Compute Monte Carlo estimate: (1/N) * sum(g(theta_i)).
4. Estimate standard error: sd(g(theta_i)) / sqrt(N).
5. Determine required N for desired precision.
6. Implement importance sampling when direct sampling is difficult.

### Verification
- MC estimate converges to known analytical result.
- Standard error decreases as 1/sqrt(N).
