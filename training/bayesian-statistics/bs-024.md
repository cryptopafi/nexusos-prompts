# BS-024 — Fit Bayesian Poisson Regression in JAGS

## Problema
Count data regression with Poisson likelihood requires careful prior specification and overdispersion handling.

## Procedura
### Prerequisites
- R
- JAGS
- BS-020 completed

### Steps
1. Write JAGS model: `y[i] ~ dpois(lambda[i])`, `log(lambda[i]) <- beta0 + beta1*x[i]`.
2. Specify priors on log-linear coefficients.
3. Prepare data: count response, predictors, sample size.
4. Run MCMC and check convergence.
5. Compute rate ratios: `exp(beta1)` with credible interval.
6. Check for overdispersion: compare observed variance to posterior predicted variance.
7. If overdispersed, extend to Negative Binomial model.

### Verification
- Model converges with adequate effective sample size.
- Rate ratios computed with credible intervals.
