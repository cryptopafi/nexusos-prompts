# BS-025 — Conduct Prior Sensitivity Analysis

## Problema
Reporting results under a single prior hides how much conclusions depend on subjective prior choices.

## Procedura
### Prerequisites
- R
- JAGS
- fitted model

### Steps
1. Identify key prior choices in your model.
2. Define a set of alternative priors: informative, weakly informative, diffuse.
3. Refit the model under each prior specification.
4. Compare posterior summaries (mean, SD, 95% CI) across prior choices.
5. Create a table or plot showing sensitivity of key parameters.
6. Assess whether substantive conclusions change across priors.
7. Document which conclusions are robust and which are prior-dependent.

### Verification
- Sensitivity table shows parameter estimates under each prior.
- Robust conclusions identified and documented.
