# BS-020 — Build Bayesian Linear Regression Model in JAGS

## Problema
Implementing regression in JAGS enables extension to non-standard likelihoods and priors.

## Procedura
### Prerequisites
- R
- JAGS
- BS-018 completed

### Steps
1. Write JAGS model: `y[i] ~ dnorm(mu[i], tau)`, `mu[i] <- beta0 + beta1*x[i]`.
2. Specify priors: `beta0 ~ dnorm(0, 0.001)`, `tau ~ dgamma(0.01, 0.01)`.
3. Prepare data list with y, x, and n.
4. Run 3 chains with different inits for 10000 iterations, 1000 burn-in.
5. Check convergence with trace plots and R-hat.
6. Summarize: posterior means and 95% credible intervals for beta0, beta1, sigma.
7. Compare with frequentist `lm()` output.

### Verification
- JAGS model converges (R-hat < 1.05).
- Posterior means close to OLS estimates.
