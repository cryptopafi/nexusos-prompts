# BS-021 — Perform Bayesian Model Checking (Residuals, DIC)

## Problema
Fitting a model without checking assumes it is correct; posterior predictive checks reveal misfit.

## Procedura
### Prerequisites
- R
- JAGS fitted model (BS-020)

### Steps
1. Generate posterior predictive samples: simulate y_rep from the fitted model.
2. Compare observed vs predicted: overlay histograms of y and y_rep.
3. Compute Bayesian p-values for key test statistics (mean, variance, max).
4. Plot residuals: observed minus posterior predictive mean.
5. Compute DIC: `dic.samples(model, n.iter=5000)`.
6. Compare DIC across competing models: lower is better.
7. Flag any systematic residual patterns indicating model misspecification.

### Verification
- Posterior predictive p-values between 0.1 and 0.9 (no extreme values).
- Residuals show no systematic pattern.
