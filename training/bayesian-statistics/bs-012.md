# BS-012 — Apply Non-Informative and Jeffreys Priors

## Problema
Objective Bayesian analysis requires priors that let the data speak, but naive flat priors can be improper.

## Procedura
### Prerequisites
- R or Python
- calculus for Fisher information

### Steps
1. Compute Jeffreys prior: pi(theta) proportional to sqrt(det(I(theta))).
2. For Binomial: derive Jeffreys prior Beta(0.5, 0.5).
3. For Normal mean: Jeffreys prior is flat (improper but posterior is proper).
4. For Normal variance: Jeffreys prior is 1/sigma^2.
5. Compare posteriors under Jeffreys vs informative priors.
6. Verify that posterior is proper even when prior is improper.

### Verification
- Jeffreys prior derived correctly for standard models.
- Posterior under Jeffreys prior matches known results.
