# BS-041 — Conduct Full Bayesian Time Series Analysis (Capstone Pipeline)

## Problema
Real-world time series analysis requires combining all Bayesian tools into a coherent analysis pipeline.

## Procedura
### Prerequisites
- R
- JAGS
- dlm package
- all prior BS procedures

### Steps
1. Explore data: plot series, compute ACF/PACF, check stationarity.
2. Fit candidate models: AR(p), NDLM with trend, NDLM with trend + season.
3. For each model: run MCMC, check convergence, compute DIC.
4. Select best model using DIC and posterior predictive checks.
5. Perform prior sensitivity analysis on the selected model.
6. Generate forecasts with full posterior uncertainty.
7. Write a complete analysis report: data description, model specification, diagnostics, results, forecasts.

### Verification
- Complete pipeline runs from data to forecast.
- Report includes all required sections with proper diagnostics.
