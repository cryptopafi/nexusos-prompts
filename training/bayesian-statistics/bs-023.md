# BS-023 — Fit Bayesian Logistic Regression in JAGS

## Problema
Binary outcome models need Bayesian treatment for proper uncertainty quantification in predictions.

## Procedura
### Prerequisites
- R
- JAGS
- BS-020 completed

### Steps
1. Write JAGS model: `y[i] ~ dbern(p[i])`, `logit(p[i]) <- beta0 + beta1*x[i]`.
2. Specify priors: `beta0 ~ dnorm(0, 0.01)`, `beta1 ~ dnorm(0, 0.01)`.
3. Prepare data: binary response, predictors, sample size.
4. Run MCMC with 3 chains, adequate iterations.
5. Check convergence (logistic models often need more iterations).
6. Compute odds ratios: `exp(beta1)` with credible interval.
7. Generate predicted probabilities for new observations.

### Verification
- Model converges (R-hat < 1.05).
- Odds ratio credible interval interpretable for decision making.
