# BS-022 — Fit Bayesian ANOVA Model in JAGS

## Problema
Frequentist ANOVA tests only null hypothesis of equal means; Bayesian ANOVA estimates group differences with uncertainty.

## Procedura
### Prerequisites
- R
- JAGS
- BS-020 completed

### Steps
1. Write JAGS model: `y[i] ~ dnorm(mu[group[i]], tau)`.
2. Define group means: `mu[j] ~ dnorm(grand_mean, tau_group)` for hierarchical ANOVA.
3. Alternative: fixed effects with separate priors per group.
4. Prepare data: response vector, group index vector, number of groups.
5. Run MCMC and check convergence.
6. Compute posterior contrasts: P(mu_1 > mu_2 | data).
7. Plot posterior distributions of group means with overlap.

### Verification
- Model converges for all group means.
- Posterior contrasts provide probabilistic group comparisons.
