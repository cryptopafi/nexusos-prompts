# BS-029 — Fit Mixture Model Using MCMC (Gibbs Sampler)

## Problema
EM gives point estimates only; MCMC quantifies uncertainty in all mixture model parameters.

## Procedura
### Prerequisites
- R or Python
- BS-017 and BS-028 completed

### Steps
1. Write the full Bayesian mixture model with priors on means, variances, and weights.
2. Augment with latent allocation variables z_i indicating component membership.
3. Derive full conditionals: z_i | rest, mu_k | rest, sigma_k | rest, pi | rest.
4. Implement Gibbs sampler cycling through all full conditionals.
5. Handle label switching: post-process with identifiability constraint (e.g., order means).
6. Run multiple chains and assess convergence.
7. Compare posterior component estimates with EM results.

### Verification
- Gibbs sampler converges for all parameters.
- Posterior means close to EM estimates.
