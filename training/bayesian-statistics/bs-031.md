# BS-031 — Apply Mixture Models for Clustering

## Problema
K-means clustering assigns hard labels; mixture models provide probabilistic cluster assignments.

## Procedura
### Prerequisites
- R or Python
- BS-028 or BS-029 completed

### Steps
1. Fit Gaussian mixture model with K components.
2. Extract posterior responsibilities (soft assignments) for each observation.
3. Assign hard clusters using MAP: `cluster[i] = argmax_k(r_ik)`.
4. Visualize clusters with scatter plot colored by assignment.
5. Compute cluster uncertainty: entropy of responsibility vector per observation.
6. Compare with K-means clustering on same data.
7. Identify observations with high uncertainty (near cluster boundaries).

### Verification
- Cluster assignments match known structure (if ground truth available).
- Uncertainty flags borderline observations.
