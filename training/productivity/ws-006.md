# WS-006 — Eliminate Multitasking and Build Focus Blocks

## Problema
Multitasking reduces productivity by 40% and increases errors, yet most knowledge workers do it habitually.

## Procedura
### Prerequisites
- WS-002 completed
- WS-004 completed

### Steps
1. Identify your peak cognitive hours (typically 2-4 hours after waking)
2. Block peak hours for single-task deep work: no meetings, no communication, one task only
3. Set up a physical or digital 'do not disturb' signal for focus blocks
4. Use the Pomodoro technique if full focus blocks feel too long: 25 minutes work + 5 minutes break
5. Close all unrelated tabs, applications, and notifications during focus blocks
6. Start each focus block with a clear single objective written down
7. Track focus block completion rate and extend duration gradually over 4 weeks

### Verification
- At least one 90-minute uninterrupted focus block per workday
- Focus block completion rate above 80% over 2 weeks
