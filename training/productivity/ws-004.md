# WS-004 — Set Up a Communication Batching Schedule

## Problema
Constant email and message checking fragments attention, with each interruption costing 15-25 minutes of refocus time.

## Procedura
### Prerequisites
- WS-002 time audit showing communication patterns

### Steps
1. Audit current communication checking frequency (email, Slack, Teams, phone)
2. Define communication batching windows: 2-3 fixed times per day for email/messages
3. Configure notification settings: disable real-time alerts during focus blocks
4. Set expectations with team: communicate your response windows and emergency escalation path
5. Use auto-responders or status messages to signal batching schedule
6. Batch similar communications: respond to all emails in one session, all messages in another
7. Measure time saved after 2 weeks of batching vs pre-batching audit

### Verification
- Communication checking reduced to 3 or fewer scheduled windows per day
- At least 1 hour of additional focus time recovered per day
