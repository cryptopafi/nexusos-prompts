# TikTok LIVE Selling Setup and Configuration

## Obiectiv
Configurarea completă a unui cont TikTok pentru live selling: eligibilitate, TikTok Shop linkare, echipamente, software, monetizare, content calendar și pre-launch testing complet, astfel încât primul live să fie capabil de vânzări directe cu rată de conversie de 5-15% (benchmark TikTok Shop).

## Pași

### 1. Verificare eligibilitate și activare LIVE
**Praguri platformă TikTok (2026)**:
- **Acces LIVE standard**: minim 1.000 followeri + cont activ 30+ zile
- **TikTok Shop LIVE selling**: 1.000+ followeri + business verificat + seller account aprobat
- **LIVE Gifts (monetizare directă)**: 1.000+ followeri, activat automat în Creator Tools
- **LIVE Subscriptions**: disponibil la 1.000+ followeri în regiunile eligibile

**Comparație praguri alte platforme**:
- **YouTube**: 1K subscriberi + 4.000 ore watch time (YPP) SAU 500 subs + 3.000 ore (YPP Lite)
- **Twitch**: 50 followeri + 8h stream/7 zile + 7 zile unice de stream + media 3 vieweri (Affiliate)
- **TikTok**: 1.000 followeri + 100K views/30 zile (LIVE monetization full)

Pași de activare:
- Profile → Creator Tools → LIVE → verifică dacă opțiunea e activă
- Dacă nu e activă: aplică prin TikTok Creator Portal (creator.tiktok.com)
- Verifică numărul de telefon și emailul (obligatoriu)
- Conturile sub 30 zile nu primesc acces LIVE indiferent de followeri

### 2. Linkare TikTok Shop la cont
- Accesează seller.tiktok.com → selectează "Europe" → creează cont de vânzător
- Selectează categoria: Food & Beverage / Lifestyle / Home & Living
- Documente necesare (EU/România): CUI activ, CI reprezentant legal, extras cont bancar pe numele firmei
- Aprobare: 1-5 zile lucrătoare; verifică statusul în Seller Center → Account Health
- Post-aprobare: linkează Shop → Settings → Linked Accounts → TikTok Shop
- Configurează metode de livrare: Seller Shipping pentru RO (DPD/Cargus/FanCourier)
- Setează politica de retur: minim 14 zile (obligatoriu EU), recomandat 30 zile (trust signal)
- Conectează cont bancar IBAN RO pentru withdrawals

### 3. Configurare echipamente live streaming
**Setup Starter (sub €150 — primele 3 luni)**:
- Camera: telefonul actual (iPhone 12+ / Samsung S21+) — 4K nativ suficient
- Stabilizare: trepied cu clemă telefon (€15-25)
- Iluminare: ring light LED 26cm ($30-80 / €25-70)
- Audio: lavalieră cu jack 3.5mm (€15-30) — audio > video ca prioritate
- Internet: minim 10 Mbps upload stabil (test pe fast.com)
- Total: €55-120

**Setup Intermediate (€300-800 — generezi venituri)**:
- Camera: Sony ZV-1 ($750 / €700) sau webcam Logitech C920 (€80)
- Capture card: Elgato Cam Link 4K ($150 / €120) pentru mirrorless → laptop
- Microfon: Rode NT-USB Mini ($170 / €100) — plug-and-play USB
- Iluminare: 2x softbox LED (€60-100 set)
- Stream controller: Elgato Stream Deck Mini ($150 / €80) — scene switching
- Total: €450-900

**Albastru/Șapte Focuri context**: bird's eye camera montată deasupra focului deschis + lavalier wireless (DJI Mic, €300) pentru libertate de mișcare. Tuciuri, cuptor pe lemne, foc deschis = vizual unic, zero competitori RO cu setup similar.

### 4. Configurare software — TikTok Studio / OBS / StreamYard
**TikTok Studio (OBLIGATORIU pentru TikTok Shop)**:
- Descarcă de pe studio.tiktok.com — stream profesional via RTMP
- Integrare nativă TikTok Shop (coșul de produse funcționează DOAR prin TikTok Studio sau app nativ)
- Setează titlul live cu keyword relevant (max 32 caractere)
- Activează "Comment Filters": Settings → Privacy → Comment Filters

**OBS Studio (pentru multi-platform, NU pentru TikTok Shop)**:
- Descarcă obsproject.com (gratuit, cross-platform)
- Setări: 1280x720, 30fps, bitrate 3000-4500 kbps, encoder x264/NVENC
- Scene: camera feed + mic + text overlay CTA + logo watermark
- IMPORTANT: TikTok Shop NU funcționează cu OBS — doar TikTok Studio sau app nativ

**StreamYard (alternativă browser-based)**:
- streamyard.com → Connect Destinations → adaugă platforme
- Zero setup tehnic, ideal pentru simulcast YouTube + Facebook

**Restream.io (simulcast management)**:
- Plan gratuit: 2 platforme; paid: nelimitat
- Chat agregat din toate platformele într-o fereastră
- RTMP URL + Stream Key → configurează în OBS

### 5. Configurare monetizare și gifting
- Activează Virtual Gifts: Creator Tools → LIVE Gifts → Enable
- Setează Diamond withdrawal: Settings → Balance → Link payment method
- Activează LIVE Subscriptions dacă ai 1K+ followeri
- Configurează "LIVE Events" pentru promovare pre-live (vizibil în FYP)

**Benchmark monetizare live streaming**:
- TikTok Shop conversie medie: 5-15% din vieweri activi
- YouTube Super Chat avg: $2-5/viewer activ
- TikTok Gifts: echivalent $0.50-2/viewer engaged
- Patreon complementar: $5 rețete, $15 box lunar, $50 clasă privată de gătit

### 6. Pregătire produse și Live Cart
- Adaugă minim 5-10 produse în Live Cart: Creator Tools → Shopping → Add to Live Cart
- Configurează prețuri live speciale în Seller Center (diferite de prețul standard)
- Creează bundle-uri: "Pachet Tuciul + Rețetă + Ingrediente" — bundle-urile cresc AOV cu 40-80%
- Testează pin/unpin fiecare produs din Live Cart
- Pregătește fizic toate produsele vizibile pe masa de live
- Creează "Product Collections" tematice: "Bucătărie tradițională", "Weekend în natură"

### 7. Content calendar — programare live-uri săptămânale
**Frecvență per stadiu**:
- Primele 30 zile: 3x/săptămână minim (algoritmul favorisează consistența)
- Maintenance: 2x/săptămână + 1 live scurt (30 min) midweek
- Scaling: 5x/săptămână (top performers TikTok Shop)

**Slot-uri orare optime (RO)**:
- Seara: 19:00-21:00 (max engagement, primetime)
- Prânz: 12:00-13:00 (audiență office/WFH)
- Weekend: 10:00-12:00 (audiență familie)

**Calendar Albastru/Șapte Focuri**:
- Marți 19:00: Live cooking rețetă tradițională (45 min)
- Joi 12:00: Quick recipe + Q&A (30 min)
- Sâmbătă 10:00: Weekend special — rețetă + vânzare pachete (60 min)

### 8. Test-run tehnic complet înainte de lansare
- Live privat 10 minute: verifică video, audio, iluminare, unghiuri
- Testează adăugarea unui produs din coș în timp real (pin product)
- Verifică delay-ul streamului (tipic 5-15 secunde pe TikTok)
- Confirmă că linkul TikTok Shop apare corect
- Testează checkout complet: cumpără un produs test din propriul live
- Checklist final: lumini OK / internet stabil / fundal curat / sunet clar / produse pregătite
- Anunță live-ul cu 1-2h înainte prin Stories/post organic
- 3-5 persoane de încredere în primele 2 minute (boost algoritm)
