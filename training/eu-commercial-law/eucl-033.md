# EUCL-033 — EU Data Governance Act (Regulation 2022/868)

## Problema
Organizations miss opportunities to participate in EU data-sharing ecosystems and fail to comply with the regulatory framework for data intermediaries and public sector data re-use.

## Procedura
### Prerequisites
- EUCL-037 (GDPR), EUCL-032 (Data Act)

### Steps
1. Understand **scope and objectives**: creates governance framework for data re-use and sharing across the EU. Complements Open Data Directive (2019/1024) and Data Act. Applies from 24 September 2023
2. Apply **re-use of protected public sector data** (Chapter II, Art. 3-9): covers public sector data subject to commercial confidentiality, statistical confidentiality, IP rights, personal data. Single information points designated by Member States. Conditions: non-exclusive, non-discriminatory, time-limited. Fees proportionate to costs. Pre-processing for anonymization/pseudonymization required for personal data
3. Register as **data intermediation service** (Chapter III, Art. 10-15) if providing: (a) intermediation between data holders and users, (b) intermediation between data subjects and users, (c) data cooperatives. Must notify national authority. Requirements: structural separation from data processing, FRAND pricing, interoperable formats, security measures, log access
4. Key **data intermediary obligations**: cannot use exchanged data for own purposes (structural neutrality), must ensure data sovereignty of data holders/subjects, cannot make data available to third parties without consent, must protect trade secrets and IP
5. Apply **data altruism** framework (Chapter IV, Art. 16-25): organizations collecting data for general interest purposes (research, public health, combating climate change, improving public services). Voluntary **European data altruism consent form** standardized by Commission. Registration as recognized data altruism organization with national authority
6. Understand the **European Data Innovation Board (EDIB)** (Chapter VI, Art. 29-30): expert group advising Commission on data governance, interoperability, cross-sectoral data sharing, standardization, international data access
7. Navigate **Common European Data Spaces** initiative: sector-specific data spaces (health, mobility, agriculture, manufacturing, energy, finance, etc.) built on DGA infrastructure. Interoperability requirements, common standards, trusted intermediaries
8. **Enforcement**: Member States designate competent authorities. Penalties for non-compliance with notification and registration requirements determined at national level but must be effective, proportionate, dissuasive

### Verification
- Can identify when a service qualifies as a data intermediation service
- Can explain the structural neutrality requirement for intermediaries
- Can describe the data altruism registration framework
