# MLT-001 — Set Up GCP AI Platform Notebooks for Financial ML

## Problema
Data scientists need a cloud-based ML environment configured for financial data analysis but struggle with GCP setup.

## Procedura
### Prerequisites
- GCP account with billing
- Basic Python knowledge

### Steps
1. Create or access a GCP project with billing enabled
2. Navigate to AI Platform > Notebooks in GCP Console
3. Create a new notebook instance with TensorFlow or PyTorch
4. Select appropriate machine type (at least n1-standard-4)
5. Install additional financial libraries (yfinance, ta-lib, pandas-datareader)
6. Verify the setup by loading a sample financial dataset

### Verification
- Notebook instance is running
- Financial libraries import without errors
- Sample data loads correctly
