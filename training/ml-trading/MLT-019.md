# MLT-019 — Calculate Hurst Exponent for Momentum Trading Signals

## Problema
The Hurst exponent reveals whether a time series is trending, mean-reverting, or random, informing strategy selection.

## Procedura
### Prerequisites
- Historical price data
- Python with numpy

### Steps
1. Collect historical price data for the target asset
2. Implement the Rescaled Range (R/S) analysis method
3. Calculate the Hurst exponent from the R/S analysis
4. Interpret: H > 0.5 indicates trending (momentum), H < 0.5 indicates mean-reversion
5. Calculate Hurst exponent over rolling windows to detect regime changes
6. Use Hurst values to switch between momentum and mean-reversion strategies

### Verification
- Hurst exponent is calculated and matches expected range (0-1)
- Rolling Hurst analysis shows regime changes over time
