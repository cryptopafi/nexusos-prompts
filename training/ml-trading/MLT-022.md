# MLT-022 — Backtest a Momentum Trading Strategy on Unseen Data

## Problema
A momentum strategy must prove itself on truly unseen data to validate its edge before committing real capital.

## Procedura
### Prerequisites
- Trained momentum model (MLT-020)
- Reserved unseen test data

### Steps
1. Reserve a final test period that was never used during development
2. Implement the full strategy with trained ML model and features
3. Run the backtest on the unseen period with realistic execution
4. Calculate performance metrics: returns, Sharpe, drawdown, turnover
5. Compare to benchmark and random strategies
6. Perform statistical significance tests on returns
7. Document the out-of-sample results honestly

### Verification
- Out-of-sample Sharpe ratio is positive
- Results are statistically significant (p < 0.05)
