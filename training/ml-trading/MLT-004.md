# MLT-004 — Forecast Stock Prices Using Linear Regression in BQML

## Problema
Linear regression provides a baseline forecasting model but requires proper feature engineering for financial data.

## Procedura
### Prerequisites
- BigQuery ML access
- Historical stock price data

### Steps
1. Collect historical price data for target stock(s)
2. Create features: lagged prices, moving averages, volume
3. Handle missing data and outliers in financial time series
4. Train a linear regression model in BQML
5. Evaluate using RMSE and MAE on held-out test data
6. Visualize predictions vs actual prices
7. Document model limitations for financial forecasting

### Verification
- Model produces predictions for future time periods
- RMSE is documented and compared to naive baseline
