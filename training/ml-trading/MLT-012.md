# MLT-012 — Implement Index Arbitrage Strategy

## Problema
Index arbitrage exploits price discrepancies between an index and its component stocks, requiring fast execution logic.

## Procedura
### Prerequisites
- Index composition data
- Historical/real-time price feeds
- Understanding of index calculation

### Steps
1. Understand the index composition and weighting methodology
2. Collect real-time or historical data for the index and all components
3. Calculate the theoretical fair value of the index from components
4. Identify deviations between market price and fair value
5. Define thresholds for trade entry (accounting for costs)
6. Simulate execution: buy cheap basket, sell expensive one
7. Backtest and measure P&L net of costs

### Verification
- Fair value calculation matches official index methodology
- Backtest identifies profitable arbitrage opportunities
