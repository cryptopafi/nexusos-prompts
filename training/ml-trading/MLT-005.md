# MLT-005 — Build an ARIMA Time Series Model for Trading

## Problema
ARIMA models capture temporal patterns in price data but require careful parameter selection and stationarity testing.

## Procedura
### Prerequisites
- Understanding of time series concepts
- BigQuery ML or Python statsmodels

### Steps
1. Load time series data and check for stationarity (ADF test)
2. Difference the data if non-stationary
3. Plot ACF and PACF to identify candidate p, d, q parameters
4. Build the ARIMA model in BQML with CREATE MODEL
5. Evaluate forecast accuracy on holdout data
6. Generate multi-step-ahead forecasts
7. Analyze residuals for remaining patterns

### Verification
- Model residuals show no significant autocorrelation
- Forecast accuracy beats naive methods
