# MLT-023 — Implement Pair Trading Strategy with Clustering

## Problema
Finding trading pairs manually is inefficient; clustering algorithms can systematically identify groups of related securities.

## Procedura
### Prerequisites
- Historical price data for 50+ stocks
- Python with sklearn

### Steps
1. Collect historical returns for a universe of stocks
2. Normalize and scale the return features
3. Apply clustering (K-means, DBSCAN, or hierarchical)
4. Identify clusters of co-moving stocks
5. Within each cluster, test pairs for cointegration
6. Select the most promising pairs based on statistical tests
7. Implement and backtest the pair trading strategy

### Verification
- Clustering produces meaningful groups of related stocks
- At least 3 cointegrated pairs are identified for trading
