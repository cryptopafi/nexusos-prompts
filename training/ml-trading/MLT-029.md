# MLT-029 — Implement Policy Iteration for Trading

## Problema
Policy iteration can converge faster than value iteration for trading MDPs with large state spaces.

## Procedura
### Prerequisites
- Trading MDP defined (MLT-028)
- Python with numpy

### Steps
1. Define the trading MDP (same framework as MLT-028)
2. Initialize a random trading policy
3. Policy Evaluation: compute value function for current policy
4. Policy Improvement: update policy greedily based on value function
5. Repeat until policy stabilizes (no changes between iterations)
6. Compare convergence speed and results with value iteration
7. Simulate the optimal policy on test data

### Verification
- Policy converges to a stable trading strategy
- Results are comparable to value iteration
