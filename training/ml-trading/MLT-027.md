# MLT-027 — Implement Kalman Filter for Trading Signal Optimization

## Problema
Static model parameters fail in non-stationary markets; the Kalman filter enables dynamic parameter estimation.

## Procedura
### Prerequisites
- Linear algebra knowledge
- Python with numpy
- Pairs trading context (MLT-024)

### Steps
1. Understand the Kalman filter state-space representation
2. Define the state equation for your trading model (e.g., hedge ratio)
3. Define the observation equation linking states to observed prices
4. Implement the Kalman filter predict-update cycle in Python
5. Apply to a pairs trading spread to get dynamic hedge ratios
6. Compare dynamic vs static hedge ratio performance
7. Tune Kalman filter noise parameters

### Verification
- Kalman filter produces smoothed, dynamic parameter estimates
- Dynamic parameters improve trading performance vs static
