# MLT-032 — Build a Deep Q-Network (DQN) for Trading

## Problema
Tabular Q-learning cannot handle continuous state spaces in trading; DQN uses neural networks to approximate Q-values.

## Procedura
### Prerequisites
- Deep learning framework (TensorFlow/PyTorch)
- Q-learning concepts (MLT-031)

### Steps
1. Define a continuous state representation (prices, indicators, position)
2. Build a neural network to approximate Q(s,a)
3. Implement experience replay buffer for stable training
4. Implement target network for reducing training oscillation
5. Train the DQN on historical trading episodes
6. Evaluate on held-out test data
7. Tune hyperparameters: batch size, replay buffer size, target update frequency

### Verification
- DQN trains stably without divergence
- Test performance exceeds tabular Q-learning baseline
