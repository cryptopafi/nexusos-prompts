# MLT-009 — Diagnose Overfitting and Underfitting in Trading Models

## Problema
Trading models that overfit produce spectacular backtest results but fail in live trading, destroying capital.

## Procedura
### Prerequisites
- A trained ML model for trading
- Train and validation datasets

### Steps
1. Split data into train, validation, and test sets (time-based splits)
2. Train a model and evaluate on all three sets
3. Plot learning curves (train vs validation loss over epochs)
4. Identify overfitting: train loss << validation loss
5. Identify underfitting: both losses are high
6. Apply remedies: regularization, dropout, more data, simpler model
7. Re-evaluate after applying remedies

### Verification
- Can diagnose whether a model is overfitting or underfitting
- Remedies improve validation performance without degrading test performance
