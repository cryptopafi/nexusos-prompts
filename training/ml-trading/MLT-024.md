# MLT-024 — Pick Trading Pairs Using Cointegration Analysis

## Problema
Pair trading requires statistically cointegrated pairs; visual correlation alone is insufficient and misleading.

## Procedura
### Prerequisites
- Candidate stock pairs
- Python with statsmodels

### Steps
1. Select candidate pairs from the same sector or cluster
2. Test for cointegration using the Engle-Granger two-step method
3. Alternatively use the Johansen test for multiple pairs
4. Calculate the spread and test for stationarity (ADF test)
5. Compute the half-life of mean reversion
6. Rank pairs by cointegration strength and half-life
7. Select pairs with half-life suitable for your trading frequency

### Verification
- Cointegration test p-values are below 0.05 for selected pairs
- Half-life calculations are consistent with trading horizon
