# MLT-028 — Implement Value Iteration for Trading MDPs

## Problema
Framing trading as a Markov Decision Process enables optimal policy computation via value iteration.

## Procedura
### Prerequisites
- Understanding of MDP framework
- Python with numpy

### Steps
1. Define the trading MDP: states, actions, transitions, rewards
2. Discretize the state space (price levels, positions, indicators)
3. Define the action space (buy, sell, hold)
4. Define the reward function (returns minus transaction costs)
5. Implement the value iteration algorithm
6. Iterate until the value function converges
7. Extract the optimal policy and simulate trades

### Verification
- Value function converges within reasonable iterations
- Extracted policy produces meaningful trading actions
