# MLT-014 — Set Up TensorFlow Environment for Financial ML

## Problema
Financial ML projects require a properly configured TensorFlow environment with financial data libraries.

## Procedura
### Prerequisites
- Python 3.8+ installed
- pip or conda package manager

### Steps
1. Install Python 3.8+ with pip or conda
2. Create a virtual environment for the project
3. Install TensorFlow 2.x: pip install tensorflow
4. Install financial libraries: yfinance, pandas, numpy, matplotlib
5. Install TA-Lib for technical analysis indicators
6. Verify installation by running a simple TF model on sample data
7. Configure GPU support if available

### Verification
- TensorFlow imports and runs without errors
- GPU is detected if available
- Financial libraries are accessible
