# MLT-033 — Implement Policy Gradient Methods for Trading

## Problema
Policy gradient methods directly optimize the trading policy, handling continuous action spaces like position sizing.

## Procedura
### Prerequisites
- Deep learning framework
- RL fundamentals

### Steps
1. Define the policy network architecture (state -> action probabilities)
2. Implement the REINFORCE algorithm with baseline
3. Define the reward function including transaction costs
4. Collect trajectories by running the policy on historical data
5. Compute policy gradient and update network
6. Implement variance reduction techniques (baseline, advantage)
7. Compare with value-based methods (DQN) on same data

### Verification
- Policy gradient converges during training
- Learned policy produces reasonable position sizing
