# MLT-037 — Optimize Portfolio Using Reinforcement Learning

## Problema
Classical portfolio optimization (Markowitz) assumes static parameters; RL enables dynamic, adaptive portfolio allocation.

## Procedura
### Prerequisites
- Multiple asset price data
- RL framework configured
- Risk management basics (MLT-036)

### Steps
1. Define the portfolio optimization as an RL problem
2. State: current allocations, market features, risk metrics
3. Action: portfolio weight adjustments
4. Reward: risk-adjusted return (Sharpe ratio based)
5. Build and train an RL agent for portfolio allocation
6. Compare RL portfolio vs Markowitz mean-variance portfolio
7. Evaluate on out-of-sample data with realistic constraints

### Verification
- RL agent produces valid portfolio allocations (sum to 1)
- Out-of-sample Sharpe ratio is competitive with Markowitz
