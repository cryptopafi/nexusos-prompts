# MLT-020 — Build a Momentum-Based Trading Model with ML

## Problema
Momentum strategies capture trends but ML can enhance signal quality and adapt to changing market conditions.

## Procedura
### Prerequisites
- Historical multi-asset price data
- ML environment configured (MLT-014)

### Steps
1. Define momentum features: returns over multiple lookback periods
2. Add supporting features: volume, volatility, sector momentum
3. Label data: future returns or direction over holding period
4. Train a classification or regression model
5. Evaluate with financial metrics: Sharpe, Sortino, max drawdown
6. Compare ML-enhanced signals vs simple momentum baseline
7. Implement risk controls (position limits, stop-losses)

### Verification
- ML model improves Sharpe ratio over simple momentum
- Risk controls are implemented and backtested
