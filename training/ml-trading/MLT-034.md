# MLT-034 — Apply LSTM Networks to Time Series Trading Data

## Problema
Standard neural networks ignore temporal dependencies; LSTMs capture sequential patterns critical for financial time series.

## Procedura
### Prerequisites
- TensorFlow/Keras or PyTorch
- Sequential financial data

### Steps
1. Prepare sequential trading data with sliding windows
2. Normalize features using rolling statistics (avoid lookahead)
3. Build an LSTM model with appropriate sequence length
4. Add dropout for regularization between LSTM layers
5. Train with early stopping on validation loss
6. Evaluate predictions on test set with financial metrics
7. Compare LSTM vs feedforward network on same data

### Verification
- LSTM trains without exploding/vanishing gradients
- Captures temporal patterns better than feedforward baseline
