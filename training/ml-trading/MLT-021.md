# MLT-021 — Create Features for Momentum Trading Models

## Problema
Raw price data is insufficient for ML models; domain-specific feature engineering is critical for momentum strategies.

## Procedura
### Prerequisites
- Historical price and volume data
- Python with pandas and ta-lib

### Steps
1. Calculate price momentum over multiple timeframes (1d, 5d, 20d, 60d)
2. Compute volume-weighted momentum indicators
3. Add volatility features (realized vol, ATR, Bollinger width)
4. Include relative strength vs benchmark and sector
5. Engineer interaction features (momentum * volume)
6. Apply proper time-series feature scaling (rolling standardization)
7. Test feature importance with correlation analysis and mutual information

### Verification
- Feature matrix is created with no lookahead bias
- Feature importance analysis identifies top predictive features
