# MLT-038 — Build Custom Models with Google Cloud AutoML

## Problema
Not all trading teams have ML expertise; AutoML democratizes model building for financial prediction tasks.

## Procedura
### Prerequisites
- GCP account with AutoML access
- Prepared financial dataset

### Steps
1. Prepare financial data in AutoML-compatible format (CSV)
2. Upload data to Google Cloud Storage
3. Create an AutoML Tables dataset
4. Configure the prediction target and feature columns
5. Launch AutoML training with budget constraints
6. Review model performance and feature importance
7. Deploy the model for online predictions
8. Compare AutoML results with manually built models

### Verification
- AutoML model trains within budget
- Performance is compared against manual baseline
