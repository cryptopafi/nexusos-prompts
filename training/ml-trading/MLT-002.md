# MLT-002 — Build a Regression Model with BigQuery ML for Finance

## Problema
Financial analysts need to build predictive models but lack ML infrastructure skills; BQML provides SQL-based model building.

## Procedura
### Prerequisites
- GCP project with BigQuery access
- Financial dataset loaded into BQ

### Steps
1. Load financial data into a BigQuery dataset
2. Explore and clean the data with SQL queries
3. Split data into training and evaluation sets
4. Create a linear regression model using CREATE MODEL syntax
5. Evaluate model performance with ML.EVALUATE
6. Make predictions with ML.PREDICT on new data

### Verification
- Model training completes without errors
- Evaluation metrics (RMSE, R-squared) are within acceptable range
