# MLT-008 — Build a Deep Neural Network for Trading with BQML

## Problema
Simple models may miss complex patterns in financial data; deep neural networks can capture non-linear relationships.

## Procedura
### Prerequisites
- BigQuery ML access
- Normalized financial feature dataset

### Steps
1. Prepare and normalize financial features for DNN input
2. Create a DNN model in BQML using CREATE MODEL with model_type='DNN_REGRESSOR'
3. Configure hidden layers, learning rate, and batch size
4. Train the model and monitor convergence
5. Evaluate with ML.EVALUATE on test data
6. Compare DNN performance against linear regression baseline
7. Analyze predictions for overfitting signals

### Verification
- DNN model trains to convergence
- Test performance is comparable to or better than linear baseline
