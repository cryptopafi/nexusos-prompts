# MLT-003 — Implement Supervised Learning for Stock Forecasting with BQML

## Problema
Predicting stock movements requires supervised learning techniques adapted to financial time series data.

## Procedura
### Prerequisites
- BigQuery ML basics (MLT-002)
- Historical stock data

### Steps
1. Prepare historical stock data with features and labels
2. Engineer features: moving averages, volume changes, RSI
3. Define the target variable (next-day price direction or return)
4. Train a classification or regression model in BQML
5. Evaluate with accuracy, precision, recall for classification
6. Analyze feature importance to understand model decisions

### Verification
- Model achieves better-than-random prediction accuracy
- Feature importance analysis provides actionable insights
