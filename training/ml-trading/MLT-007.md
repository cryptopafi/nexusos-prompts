# MLT-007 — Forecast Stock Prices Using ARIMA in BigQuery ML

## Problema
Practitioners need a cloud-scalable ARIMA implementation for stock price forecasting without managing infrastructure.

## Procedura
### Prerequisites
- BigQuery ML access
- Stock price data in BigQuery

### Steps
1. Prepare stock price time series in BigQuery
2. Create an ARIMA_PLUS model using BigQuery ML
3. Configure auto-ARIMA for automatic parameter selection
4. Evaluate model fit with ML.EVALUATE and ML.ARIMA_EVALUATE
5. Generate forecasts with ML.FORECAST
6. Export forecasts and visualize against actual prices

### Verification
- ARIMA model trains successfully in BQML
- Forecasts are generated for the specified horizon
