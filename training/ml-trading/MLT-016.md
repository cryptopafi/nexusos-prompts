# MLT-016 — Build Neural Networks with Keras Functional API for Trading

## Problema
Complex trading models with multiple inputs (price, volume, sentiment) require the flexibility of the Functional API.

## Procedura
### Prerequisites
- Keras Sequential experience (MLT-015)
- Multi-source financial data

### Steps
1. Define multiple input layers for different data types
2. Build separate processing branches for each input
3. Merge branches with concatenate or attention layers
4. Add shared dense layers after merge
5. Compile and train with multiple input data generators
6. Evaluate and compare against Sequential model baseline
7. Visualize the model architecture with plot_model

### Verification
- Multi-input model trains without errors
- Performance meets or exceeds single-input baseline
