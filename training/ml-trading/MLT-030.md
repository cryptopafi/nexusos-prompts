# MLT-030 — Implement TD-Learning for Online Trading

## Problema
Model-free TD-learning enables trading agents to learn from experience without needing a complete market model.

## Procedura
### Prerequisites
- Understanding of TD-learning theory
- Trading environment (simulated or historical)

### Steps
1. Define state representation for the trading environment
2. Initialize value estimates for all states
3. Implement TD(0) update rule: V(s) += alpha * (r + gamma*V(s') - V(s))
4. Run episodes of trading and update values online
5. Tune learning rate (alpha) and discount factor (gamma)
6. Track value estimates convergence over episodes
7. Compare learned values with Monte Carlo estimates

### Verification
- Value estimates converge over episodes
- Learned values produce reasonable trading signals
