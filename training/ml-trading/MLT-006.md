# MLT-006 — Choose ARIMA Parameters Using Auto-Correlation Analysis

## Problema
Wrong ARIMA parameters lead to poor forecasts; systematic ACF/PACF analysis is required for optimal selection.

## Procedura
### Prerequisites
- Financial time series data
- Python with statsmodels or equivalent

### Steps
1. Plot the raw time series and identify trends/seasonality
2. Compute and plot the Autocorrelation Function (ACF)
3. Compute and plot the Partial Autocorrelation Function (PACF)
4. Identify the AR order (p) from PACF cutoff
5. Identify the MA order (q) from ACF cutoff
6. Determine differencing order (d) from stationarity tests
7. Validate parameters with AIC/BIC comparison

### Verification
- Selected parameters produce lower AIC/BIC than alternatives
- Model diagnostics confirm appropriate parameter choice
