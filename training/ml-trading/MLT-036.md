# MLT-036 — Implement Portfolio Risk Management with ML

## Problema
Traditional risk models use static assumptions; ML can capture dynamic risk factors and tail events more accurately.

## Procedura
### Prerequisites
- Portfolio data
- Python with scipy, sklearn

### Steps
1. Collect portfolio holdings and historical returns
2. Calculate traditional risk metrics: VaR, CVaR, volatility
3. Build an ML model to predict portfolio volatility (GARCH or neural network)
4. Implement dynamic position sizing based on predicted risk
5. Stress test the portfolio with historical crisis scenarios
6. Build a risk dashboard with real-time monitoring
7. Set up alerts for risk threshold breaches

### Verification
- Risk model produces VaR and CVaR estimates
- Dynamic position sizing reduces drawdowns in backtest
