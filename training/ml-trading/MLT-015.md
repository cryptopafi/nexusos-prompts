# MLT-015 — Build Neural Networks with Keras Sequential API for Trading

## Problema
The Keras Sequential API provides a straightforward way to build feedforward networks for trading signal prediction.

## Procedura
### Prerequisites
- TensorFlow/Keras installed (MLT-014)
- Prepared financial feature dataset

### Steps
1. Prepare and normalize financial features (returns, volume, indicators)
2. Define a Sequential model with Dense layers
3. Add activation functions (ReLU for hidden, sigmoid/softmax for output)
4. Compile with appropriate loss function (MSE for regression, cross-entropy for classification)
5. Train with early stopping and validation split
6. Evaluate on held-out test data
7. Interpret model predictions for trading decisions

### Verification
- Model trains and converges
- Test set performance is documented and compared to baseline
