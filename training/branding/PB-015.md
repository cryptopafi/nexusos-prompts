# PB-015 — Create a Production Plan for Brand Campaign

## Problema
Campaigns stall or go over budget without a detailed production plan that accounts for all deliverables, timelines, and dependencies.

## Procedura
### Prerequisites
- Approved creative concept (PB-013, PB-014)
- Budget allocation

### Steps
1. List all campaign deliverables: hero assets, social cuts, print adaptations, digital banners, etc.
2. For each deliverable, specify format, dimensions, duration, and platform requirements.
3. Create a production timeline with milestones: brief sign-off, pre-production, shoot/design, post-production, QA, delivery.
4. Assign owners and vendors for each deliverable.
5. Build the production budget: talent, crew, equipment, location, post-production, licensing, contingency (10%).
6. Identify critical path items and dependencies between deliverables.
7. Set up a review and approval workflow with stakeholder sign-off gates.
8. Create a risk register: what could go wrong and what is the mitigation plan.

### Verification
- All deliverables listed with specs
- Timeline includes dependencies and critical path
- Budget includes 10% contingency
