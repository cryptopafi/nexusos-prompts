# PB-020 — Set Up CX Metrics and Dashboard

## Problema
Without a unified CX dashboard, teams rely on anecdotal feedback and miss systemic issues that erode brand perception over time.

## Procedura
### Prerequisites
- Customer journey map (PB-016)
- Access to analytics, CRM, and support tools

### Steps
1. Select core CX metrics: NPS (loyalty), CSAT (satisfaction), CES (effort), and churn rate.
2. Map each metric to a customer journey stage to identify where each is most relevant.
3. Set up data collection: post-interaction surveys (CSAT/CES), periodic relationship surveys (NPS), and behavioral tracking (churn).
4. Build the dashboard with three views: executive summary, journey-stage detail, and trend over time.
5. Set alert thresholds: what metric level triggers investigation? (e.g., NPS drops 5+ points).
6. Establish reporting cadence: weekly operational, monthly strategic, quarterly board-level.
7. Run the first full measurement cycle and baseline all metrics.

### Verification
- All 4 core metrics defined with collection methods
- Dashboard built with 3 views
- Baseline measurements completed
