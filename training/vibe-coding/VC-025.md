# VC-025 — Use Copilot for Pull Requests and Code Reviews

## Problema
Code reviews are time-consuming and Copilot can help summarize PRs and suggest improvements.

## Procedura
### Prerequisites
- GitHub Copilot subscription
- Repository on GitHub with PRs

### Steps
1. Open a pull request on GitHub
2. Enable Copilot PR summary generation
3. Review the AI-generated PR description
4. Use Copilot to review code changes for issues
5. Request Copilot suggestions for improvement
6. Incorporate feedback and update the PR

### Verification
- PR has an AI-generated summary
- Code review suggestions are actionable
