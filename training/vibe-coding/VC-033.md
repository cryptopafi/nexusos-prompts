# VC-033 — Use Slash Commands in Claude Code

## Problema
Users underutilize Claude Code's built-in slash commands that speed up common development tasks.

## Procedura
### Prerequisites
- Claude Code installed

### Steps
1. Type / in Claude Code to see available commands
2. Use /help to explore all commands
3. Use /compact to reduce context and free up memory
4. Use /plan to create execution plans
5. Use /undo to revert changes
6. Use /status to check current session state

### Verification
- Can use at least 5 slash commands from memory
- Understand when each command is most useful
