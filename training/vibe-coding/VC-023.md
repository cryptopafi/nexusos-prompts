# VC-023 — Build a Complete Project End-to-End with GitHub Copilot

## Problema
Learners need a structured exercise to practice using all Copilot features together in a real project workflow.

## Procedura
### Prerequisites
- All Copilot features understood (VC-013 through VC-022)
- Node.js or Python installed

### Steps
1. Define a small project (e.g., a REST API with CRUD operations)
2. Use Agent Mode to scaffold the project
3. Use inline completions to speed up implementation
4. Use Chat Mode to debug issues
5. Use Edit Mode for targeted refactoring
6. Use /tests to generate test coverage
7. Use /doc to document the codebase
8. Deploy and verify the complete application

### Verification
- Project is fully functional with tests
- Code is documented
- All Copilot modes were used during development
