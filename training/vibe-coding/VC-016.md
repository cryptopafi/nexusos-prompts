# VC-016 — Use GitHub Copilot Ask/Chat Mode for Code Q&A

## Problema
Developers waste time searching documentation when Copilot Chat can answer code questions with project-aware context.

## Procedura
### Prerequisites
- GitHub Copilot installed (VC-013)

### Steps
1. Open the Copilot Chat panel in VS Code
2. Ask a question about your current code (e.g., 'What does this function do?')
3. Ask for explanations of error messages
4. Request code examples for a specific pattern
5. Use @workspace to ask about the entire project
6. Save useful responses for future reference

### Verification
- Chat provides accurate answers about your code
- Answers include relevant code examples
