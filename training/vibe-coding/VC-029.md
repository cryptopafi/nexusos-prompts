# VC-029 — Write Claude Rules and Use Plan Mode

## Problema
Claude Code generates inconsistent code when developers don't define project rules or use plan mode for complex tasks.

## Procedura
### Prerequisites
- Claude Code installed (VC-027)
- A project to work on

### Steps
1. Create a CLAUDE.md file in your project root
2. Define coding standards and conventions
3. Specify technology stack and preferences
4. Use /plan to create an execution plan before coding
5. Review and approve the plan
6. Execute the plan with Claude Code

### Verification
- CLAUDE.md is created and followed by Claude
- Plans produce structured, reviewable execution steps
