# VC-040 — Identify Automatable Tasks for AI App Development

## Problema
Developers don't know how to systematically identify which of their daily tasks are good candidates for AI automation.

## Procedura
### Prerequisites
- Pen and paper or a spreadsheet

### Steps
1. List all recurring tasks you perform weekly
2. Rate each task on repetitiveness (1-5)
3. Rate each task on rule-based logic (1-5)
4. Rate each task on data availability (1-5)
5. Score and rank tasks by automation potential
6. Select the top 3 candidates for AI app development

### Verification
- A ranked list of automatable tasks exists
- Top 3 candidates have clear automation specifications
