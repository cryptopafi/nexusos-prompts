# VC-015 — Use GitHub Copilot Edit Mode for Code Modification

## Problema
Developers need to modify existing code precisely but inline suggestions are too broad for targeted edits.

## Procedura
### Prerequisites
- GitHub Copilot installed
- Existing code to modify

### Steps
1. Select the code block you want to modify
2. Activate Copilot Edit mode (Cmd+I / Ctrl+I)
3. Describe the desired change in natural language
4. Review the proposed diff before accepting
5. Accept or reject the changes
6. Test that the modification works correctly

### Verification
- Code is modified as requested
- No unintended changes to surrounding code
