# VC-017 — Use Inline Code Completion with GitHub Copilot

## Problema
Developers don't know the keyboard shortcuts and techniques to get the best inline suggestions from Copilot.

## Procedura
### Prerequisites
- GitHub Copilot installed and active

### Steps
1. Open a code file in VS Code with Copilot active
2. Start typing a function signature and pause for suggestions
3. Use Tab to accept a full suggestion
4. Use Cmd+Right to accept word-by-word
5. Use Alt+] and Alt+[ to cycle through alternative suggestions
6. Write descriptive comments to guide better completions

### Verification
- Can accept, reject, and cycle through suggestions fluently
- Suggestions are contextually relevant to your code
