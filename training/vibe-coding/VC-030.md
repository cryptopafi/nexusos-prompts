# VC-030 — Use Checkpoints and Security Checks in Claude Code

## Problema
Developers risk losing work or introducing vulnerabilities without understanding Claude Code's safety features.

## Procedura
### Prerequisites
- Claude Code installed and configured

### Steps
1. Understand how Claude Code creates automatic checkpoints
2. Use git integration to track AI-generated changes
3. Review the permission prompt before file modifications
4. Configure allowed and denied commands in settings
5. Use /undo to revert to a previous checkpoint
6. Review security warnings for shell commands

### Verification
- Can revert to a checkpoint after unwanted changes
- Understand which commands require approval
