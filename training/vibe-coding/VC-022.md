# VC-022 — Use Slash Commands in GitHub Copilot

## Problema
Users miss productivity shortcuts available through Copilot's slash commands for common coding tasks.

## Procedura
### Prerequisites
- GitHub Copilot installed

### Steps
1. Open Copilot Chat in VS Code
2. Type / to see available slash commands
3. Use /explain to get code explanations
4. Use /fix to get bug fix suggestions
5. Use /tests to generate unit tests
6. Use /doc to generate documentation
7. Use /new to scaffold new projects

### Verification
- Can use at least 5 slash commands from memory
- Each command produces useful output
