# VC-028 — Configure Claude Code Plans and Pricing

## Problema
Users are confused about Claude Code pricing tiers and which plan suits their usage level.

## Procedura
### Prerequisites
- Claude Code installed (VC-027)

### Steps
1. Review Claude Code pricing at anthropic.com
2. Understand the difference between API-based and Max subscription usage
3. Calculate estimated monthly cost based on your usage patterns
4. Choose between Pro, Max, or API billing
5. Set up billing and usage limits
6. Monitor usage with the /usage command

### Verification
- Billing is configured correctly
- Usage limits are set to prevent surprises
