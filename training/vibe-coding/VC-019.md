# VC-019 — Build Features with Copilot Agent Mode

## Problema
Using Agent Mode for real feature development requires understanding how to scope tasks and review autonomous changes.

## Procedura
### Prerequisites
- GitHub Copilot Agent Mode (VC-018)
- Existing project with test suite

### Steps
1. Define a clear feature specification
2. Break the feature into logical sub-tasks
3. Open Copilot Agent Mode and describe the first sub-task
4. Review the agent's plan before execution
5. Let the agent implement and test
6. Review the generated code diff
7. Move to the next sub-task, providing context from previous steps
8. Run the full test suite after all sub-tasks complete

### Verification
- Feature is fully implemented and functional
- All existing tests still pass
