# VC-035 — Build a Calendar App with Claude Code (Full Project)

## Problema
Learners need a guided full-project exercise to practice Claude Code's complete feature set in a realistic scenario.

## Procedura
### Prerequisites
- Claude Code installed and configured
- Node.js or Python runtime

### Steps
1. Define calendar app requirements (add/edit/delete events, views, reminders)
2. Create a CLAUDE.md with project rules and tech stack
3. Use /plan to outline the implementation approach
4. Build the backend API with Claude Code
5. Build the frontend UI with Claude Code
6. Add event CRUD functionality
7. Implement calendar views (day, week, month)
8. Test all features and fix bugs with AI assistance

### Verification
- Calendar app runs locally with all features
- Events persist across sessions
- All views render correctly
