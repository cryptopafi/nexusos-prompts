# VC-026 — Set Up MCP with GitHub Copilot

## Problema
Copilot's capabilities are limited without MCP servers that connect it to external tools and data sources.

## Procedura
### Prerequisites
- GitHub Copilot with Agent Mode
- MCP concepts (VC-011)

### Steps
1. Understand MCP basics (VC-011)
2. Identify an MCP server to integrate (e.g., filesystem, database)
3. Configure the MCP server in VS Code settings
4. Restart VS Code to activate the MCP connection
5. Test the MCP tool from Copilot Agent Mode
6. Verify data flows correctly between Copilot and the MCP server

### Verification
- MCP server connects successfully
- Copilot can invoke MCP tools and use the results
