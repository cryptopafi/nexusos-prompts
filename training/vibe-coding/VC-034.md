# VC-034 — Develop with Claude Code Multi-Agent Workflows

## Problema
Complex projects benefit from multiple Claude Code agents working in parallel but developers lack patterns for orchestration.

## Procedura
### Prerequisites
- Claude Code installed
- Git proficiency
- Large project that benefits from parallelization

### Steps
1. Understand the concept of multi-agent workflows
2. Set up a main orchestrator agent with a CLAUDE.md plan
3. Spawn sub-agents for parallel tasks using separate terminals
4. Define clear boundaries and interfaces between agents
5. Use git branches to isolate each agent's work
6. Merge agent outputs and resolve conflicts
7. Test the integrated result

### Verification
- Multiple agents complete their tasks independently
- Merged result is functional and conflict-free
