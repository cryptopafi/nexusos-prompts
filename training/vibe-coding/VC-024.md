# VC-024 — Configure Custom Instructions for GitHub Copilot

## Problema
Default Copilot behavior doesn't align with project-specific conventions, resulting in code that needs manual adjustment.

## Procedura
### Prerequisites
- GitHub Copilot installed
- Understanding of project conventions

### Steps
1. Create a .github/copilot-instructions.md file in your repo
2. Define coding style and conventions
3. Specify framework and library preferences
4. Add project-specific patterns and anti-patterns
5. Test that Copilot follows the instructions
6. Iterate on instructions based on output quality

### Verification
- Copilot-generated code follows custom instructions
- Instructions file is version-controlled
