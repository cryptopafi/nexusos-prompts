# VC-027 — Install and Set Up Claude Code CLI

## Problema
Developers want to use Claude Code but need step-by-step installation and initial configuration guidance.

## Procedura
### Prerequisites
- Node.js 18+ installed
- Anthropic account with API access or Max subscription

### Steps
1. Ensure Node.js 18+ is installed
2. Install Claude Code globally: npm install -g @anthropic-ai/claude-code
3. Run claude in your terminal to start
4. Authenticate with your Anthropic account
5. Verify the installation by asking a simple question
6. Explore the help menu with /help

### Verification
- claude command works in terminal
- Authentication is successful
