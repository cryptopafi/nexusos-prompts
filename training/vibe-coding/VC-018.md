# VC-018 — Use GitHub Copilot Agent Mode for Autonomous Coding

## Problema
Developers want Copilot to handle multi-step coding tasks autonomously but don't know how to activate and use Agent Mode.

## Procedura
### Prerequisites
- GitHub Copilot with Agent Mode access
- VS Code latest version

### Steps
1. Ensure you have VS Code Insiders or latest stable with Agent Mode support
2. Open the Copilot Chat panel
3. Switch to Agent Mode using the mode selector
4. Provide a multi-step task description
5. Observe the agent planning and executing steps
6. Review terminal commands the agent wants to run
7. Approve or reject each action
8. Verify the final result

### Verification
- Agent completes the multi-step task
- All generated code compiles and runs
