# VC-036 — Set Up MCP Servers with Claude Code

## Problema
Claude Code's power is limited without MCP servers providing access to external tools, databases, and APIs.

## Procedura
### Prerequisites
- Claude Code installed
- MCP concepts understood (VC-011)

### Steps
1. Review available MCP servers in the MCP registry
2. Choose an MCP server relevant to your workflow
3. Add the MCP server configuration to Claude Code settings
4. Restart Claude Code to activate the MCP connection
5. Test the MCP tools by invoking them in conversation
6. Build a workflow that combines multiple MCP tools

### Verification
- MCP server connects and tools are available
- Claude Code can use MCP tools to complete tasks
