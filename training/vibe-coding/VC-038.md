# VC-038 — Build a Custom AI Tool Using Google AI Studio Vibe Coding

## Problema
Developers want to experiment with Google's AI models for vibe coding but lack a starting workflow.

## Procedura
### Prerequisites
- Google account
- Web browser

### Steps
1. Open Google AI Studio in your browser
2. Explore the model playground and available models
3. Define a simple tool concept (e.g., text summarizer, code explainer)
4. Build the tool using the AI Studio interface
5. Test with various inputs
6. Export or deploy the tool for regular use

### Verification
- Custom tool works as intended
- Tool handles edge cases gracefully
