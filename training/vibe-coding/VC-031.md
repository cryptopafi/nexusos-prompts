# VC-031 — Add Images as Context in Claude Code

## Problema
Developers don't know they can share screenshots and mockups with Claude Code to guide visual implementation.

## Procedura
### Prerequisites
- Claude Code installed
- Image files to use as context

### Steps
1. Prepare an image file (screenshot, mockup, or diagram)
2. Use the image paste feature in Claude Code terminal
3. Reference the image in your prompt for context
4. Ask Claude to implement a UI based on the image
5. Compare the generated output to the original image
6. Iterate with additional images for refinement

### Verification
- Claude accurately interprets the image content
- Generated code matches the visual reference
