# VC-041 — Evaluate Workflow Suitability for AI App Building

## Problema
Not every workflow benefits from AI automation; developers need a framework to evaluate suitability before investing time.

## Procedura
### Prerequisites
- A workflow to evaluate
- Completion of VC-040

### Steps
1. Define the current workflow end-to-end
2. Identify steps that require human judgment vs mechanical execution
3. Assess data format consistency (structured vs unstructured)
4. Evaluate error tolerance and failure consequences
5. Calculate estimated development effort vs time savings
6. Make a go/no-go decision based on the evaluation

### Verification
- Evaluation document with clear go/no-go recommendation
- Decision is backed by quantified analysis
