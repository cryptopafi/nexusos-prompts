# VC-021 — Use Context Tools in GitHub Copilot (@workspace, @file)

## Problema
Copilot answers are generic when users don't leverage context tools to ground responses in their actual codebase.

## Procedura
### Prerequisites
- GitHub Copilot installed
- Multi-file project open

### Steps
1. Open a multi-file project in VS Code
2. Use @workspace in Copilot Chat to reference the entire project
3. Use @file to reference a specific file
4. Use #selection to reference highlighted code
5. Combine context tools for precise queries
6. Compare answer quality with and without context tools

### Verification
- Copilot answers reference actual project code
- Suggestions integrate correctly with existing architecture
