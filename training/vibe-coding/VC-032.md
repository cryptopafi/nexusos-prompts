# VC-032 — Configure Claude Code Hooks for Automation

## Problema
Manual repetitive tasks in the Claude Code workflow can be automated with hooks but developers don't know how to set them up.

## Procedura
### Prerequisites
- Claude Code installed
- Familiarity with JSON configuration

### Steps
1. Understand the hook system in Claude Code settings
2. Create a pre-tool-use hook for validation
3. Create a post-tool-use hook for logging or formatting
4. Configure notification hooks for long-running tasks
5. Test hooks with a sample workflow
6. Debug hooks using the Claude Code output panel

### Verification
- Hooks fire correctly at the configured trigger points
- Automated tasks execute without errors
