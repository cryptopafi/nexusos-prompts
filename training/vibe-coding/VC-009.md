# VC-009 — Build a Quote Generator App with Cursor AI (Full Project)

## Problema
Learners complete tutorial snippets but never build a complete app end-to-end with AI, missing the experience of full-cycle vibe coding: spec → scaffold → features → polish → deploy.

## Procedura
### Prerequisites
- Cursor AI in Agent Mode (VC-004)
- Node.js 18+ installed
- Basic understanding of HTML/CSS/JavaScript

### Steps
1. **Define the app spec** — Quote Generator app: displays random quotes, allows favoriting, stores favorites in localStorage, responsive UI, deployable to Vercel/Netlify. Tech: React + TypeScript + Tailwind CSS + Vite.
2. **Scaffold with Agent** — `Cmd+I` → Agent → prompt: "Create a React + TypeScript + Tailwind + Vite project called quote-generator. Set up the folder structure: src/components/, src/hooks/, src/types/, src/data/. Include a quotes.json with 50 motivational quotes (author + text)."
3. **Build the QuoteCard component** — prompt: "Create a QuoteCard component that displays a quote with the author name, a favorite button (heart icon), and a share button. Use Tailwind for styling — dark background, white text, rounded corners, subtle shadow."
4. **Add state management** — prompt: "Create a useQuotes hook that: (a) loads quotes from quotes.json, (b) provides a getRandomQuote function, (c) manages a favorites array stored in localStorage, (d) provides toggleFavorite function."
5. **Build the main layout** — prompt: "Create the App component with: a header showing 'Daily Quotes', the QuoteCard centered on screen, a 'New Quote' button that triggers getRandomQuote, and a 'Favorites' sidebar that slides in from the right showing saved quotes."
6. **Add animations** — prompt: "Add Framer Motion animations: fade-in on quote change, scale-up on favorite button click, slide-in for the favorites sidebar. Install framer-motion and implement."
7. **Add keyboard shortcuts** — prompt: "Add keyboard shortcuts: Space = new quote, F = toggle favorite, Escape = close sidebar. Use a useKeyboard hook. Show shortcut hints in the UI footer."
8. **Make responsive** — prompt: "Make the app fully responsive: mobile-first layout, sidebar becomes a bottom sheet on mobile, quote text scales with viewport width. Test at 320px, 768px, and 1440px widths."
9. **Add tests** — prompt: "Write Vitest tests for the useQuotes hook (load, random, favorite, localStorage persistence) and the QuoteCard component (render, click handlers). Target 80%+ coverage."
10. **Deploy** — prompt: "Add a Vercel deployment config (vercel.json), update README.md with setup instructions and a screenshot placeholder. Run build and verify zero errors." Then deploy: `npx vercel --prod`.

### Verification
- [ ] App runs locally with `npm run dev` — zero errors
- [ ] Random quote changes on button click and Space key
- [ ] Favorites persist across browser refresh (localStorage)
- [ ] Responsive at all 3 breakpoints (320px, 768px, 1440px)
- [ ] Tests pass with 80%+ coverage
- [ ] Deployed URL accessible and functional
