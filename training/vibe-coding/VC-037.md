# VC-037 — Plan and Execute Solo AI Coding Projects

## Problema
Solo developers lack a structured methodology for planning and executing projects entirely with AI coding tools.

## Procedura
### Prerequisites
- At least one AI coding tool set up (Cursor, Copilot, or Claude Code)

### Steps
1. Define the project scope and requirements clearly
2. Choose the right AI coding tool for the project
3. Create a project brief document
4. Break the project into milestones with deliverables
5. Use AI to scaffold and implement each milestone
6. Test each milestone before proceeding
7. Document decisions and lessons learned

### Verification
- Project is completed within the planned milestones
- Documentation captures the development process
