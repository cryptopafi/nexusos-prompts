# VC-020 — Manage Chat Context and Start New Chats Effectively

## Problema
Long chat threads confuse the AI with stale context, degrading response quality over time.

## Procedura
### Prerequisites
- GitHub Copilot or Cursor AI chat feature

### Steps
1. Observe how response quality degrades in long chat threads
2. Start a new chat session when switching tasks
3. Use @file to provide fresh context in new chats
4. Keep prompts focused on a single topic per chat
5. Use chat history to reference previous solutions
6. Establish a habit of resetting context every 10-15 exchanges

### Verification
- Response quality remains high across sessions
- Can efficiently manage multiple chat contexts
