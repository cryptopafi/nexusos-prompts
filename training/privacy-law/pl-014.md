# PL-014 — Implement California Right-to-Delete Procedures

## Problema
California's right to delete requires specific technical and operational capabilities that many organizations lack.

## Procedura
### Prerequisites
- PL-001 completed
- Understanding of CCPA/CPRA requirements

### Steps
1. Establish intake channels for deletion requests (toll-free number, website, email)
2. Implement identity verification process (two-step for non-account holders)
3. Map all systems containing the consumer's personal information
4. Execute deletion across all identified systems including backups (or document exceptions)
5. Notify service providers and contractors to delete consumer's data
6. Apply statutory exceptions: legal obligation, security, exercise of rights, internal uses compatible with expectations
7. Confirm deletion to the consumer within 45 days (one 45-day extension with notice)
8. Maintain deletion request log for compliance documentation

### Verification
- Deletion executed across all mapped systems
- Response provided within 45-day statutory timeline
