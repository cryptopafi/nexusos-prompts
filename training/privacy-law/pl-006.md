# PL-006 — Apply HIPAA Minimum Necessary Standard

## Problema
Over-sharing PHI internally and with business associates is a pervasive compliance failure that the minimum necessary standard addresses.

## Procedura
### Prerequisites
- PL-003 completed

### Steps
1. Identify which uses and disclosures are subject to minimum necessary (excludes treatment, patient requests, legal mandates)
2. Define role-based access policies: which workforce members need which PHI categories
3. Implement technical access controls reflecting role-based policies
4. Establish protocols for routine vs non-routine disclosures
5. Create reasonable reliance criteria for requests from other covered entities
6. Train workforce on minimum necessary principles
7. Audit access logs for minimum necessary compliance

### Verification
- Role-based access matrix documented for all workforce categories
- Audit demonstrates no systematic over-access patterns
