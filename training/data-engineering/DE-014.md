# DE-014 — Work with Python IDE for Data Engineering

## Problema
Choosing and configuring the right IDE increases productivity for data engineering development tasks.

## Procedura
### Prerequisites
- Python installed
- IDE downloaded

### Steps
1. Install VS Code or PyCharm
2. Configure Python interpreter and virtual environment
3. Install data engineering extensions (Python, Jupyter, SQL)
4. Set up linting and formatting (pylint, black)
5. Configure integrated terminal for running scripts
6. Set up debugging with breakpoints

### Verification
- IDE is configured with Python interpreter
- Linting and formatting work on save
