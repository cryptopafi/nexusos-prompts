# DE-073 — Evaluate and Tune ML Models in Spark

## Problema
Selecting optimal hyperparameters in distributed ML requires systematic tuning with cross-validation.

## Procedura
### Prerequisites
- Spark ML model to tune (DE-070 or DE-071)

### Steps
1. Define a parameter grid with ParamGridBuilder
2. Set up CrossValidator with estimator, evaluator, and param grid
3. Run cross-validation to find best parameters
4. Retrieve the best model and its parameters
5. Evaluate the best model on held-out test data
6. Compare cross-validated performance vs default parameters
7. Document the tuning results and final model selection

### Verification
- Cross-validation completes for all parameter combinations
- Best model outperforms default parameters
