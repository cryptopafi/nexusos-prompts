# DE-027 — Sort and Group SQL Result Sets

## Problema
Sorting and grouping transform raw data into organized, summarized information for business decision-making.

## Procedura
### Prerequisites
- Database with sample data

### Steps
1. Use ORDER BY with ASC and DESC
2. Sort by multiple columns
3. Use GROUP BY to aggregate data
4. Apply aggregate functions: SUM, AVG, MIN, MAX, COUNT
5. Use HAVING to filter grouped results
6. Combine GROUP BY with ORDER BY for sorted summaries

### Verification
- Results are sorted correctly
- GROUP BY produces accurate aggregates
- HAVING filters groups correctly
