# DE-022 — Set Up and Configure PostgreSQL Database

## Problema
PostgreSQL is the preferred open-source database for advanced features; proper setup ensures optimal performance.

## Procedura
### Prerequisites
- Admin access to the machine

### Steps
1. Install PostgreSQL (brew install postgresql or apt install postgresql)
2. Initialize the database cluster if needed
3. Create a database user with createuser
4. Create a database with createdb
5. Configure postgresql.conf: shared_buffers, work_mem
6. Configure pg_hba.conf for authentication
7. Test connection with psql CLI

### Verification
- PostgreSQL server is running
- User can connect with psql and run queries
