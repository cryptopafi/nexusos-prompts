# DE-013 — Build an ETL Pipeline in Python

## Problema
Manual data processing is error-prone and not scalable; ETL pipelines automate data flow from source to destination.

## Procedura
### Prerequisites
- Python with pandas
- Source and destination systems accessible

### Steps
1. Define source data and destination schema
2. Extract: read data from source (file, API, database)
3. Transform: clean, validate, and reshape data
4. Load: insert data into the destination (database, warehouse)
5. Add error handling and logging
6. Implement incremental loading (process only new/changed data)
7. Schedule the pipeline to run automatically

### Verification
- Pipeline runs end-to-end without errors
- Destination contains correctly transformed data
