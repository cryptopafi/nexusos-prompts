# DE-070 — Build ML Classification Models with Spark MLlib

## Problema
Big data classification requires distributed ML; Spark MLlib scales to datasets that don't fit in memory.

## Procedura
### Prerequisites
- PySpark with MLlib
- Labeled dataset

### Steps
1. Prepare a labeled dataset in Spark DataFrame format
2. Create feature vectors using VectorAssembler
3. Split data into train and test sets
4. Train a classification model (LogisticRegression, RandomForestClassifier)
5. Make predictions on the test set
6. Evaluate with MulticlassClassificationEvaluator
7. Save the model for production use

### Verification
- Model trains on the full distributed dataset
- Evaluation metrics (accuracy, F1) are documented
