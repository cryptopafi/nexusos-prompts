# DE-071 — Build ML Regression Models with Spark MLlib

## Problema
Predicting continuous values at scale requires distributed regression models available in Spark MLlib.

## Procedura
### Prerequisites
- PySpark with MLlib
- Dataset with continuous target

### Steps
1. Prepare a dataset with continuous target variable
2. Engineer features and create feature vectors
3. Split into train and test sets
4. Train a regression model (LinearRegression, GBTRegressor)
5. Predict on test data
6. Evaluate with RegressionEvaluator (RMSE, R2, MAE)
7. Compare multiple models and select the best

### Verification
- Model produces predictions for continuous values
- RMSE and R2 are documented and compared across models
