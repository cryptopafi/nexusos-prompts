# DE-029 — Join Multiple Tables

## Problema
Real-world data is spread across multiple tables; joins combine them into meaningful analytical datasets.

## Procedura
### Prerequisites
- Database with related tables (foreign keys)

### Steps
1. Write INNER JOIN to combine matching rows
2. Write LEFT JOIN to keep all rows from the left table
3. Write RIGHT JOIN to keep all rows from the right table
4. Write FULL OUTER JOIN to keep all rows from both tables
5. Write CROSS JOIN for Cartesian products
6. Join three or more tables in a single query
7. Use table aliases for readability in multi-join queries

### Verification
- Each join type returns the correct result set
- Multi-table joins produce accurate combined data
