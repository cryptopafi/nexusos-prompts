# DE-024 — Write Basic SQL Queries

## Problema
SQL is the universal language for data; engineers must write queries fluently for any data task.

## Procedura
### Prerequisites
- Database with tables and sample data

### Steps
1. Write SELECT statements to retrieve specific columns
2. Use WHERE clause to filter rows
3. Use ORDER BY to sort results
4. Use aliases for columns and tables
5. Insert data with INSERT INTO
6. Update data with UPDATE ... SET ... WHERE
7. Delete data with DELETE FROM ... WHERE

### Verification
- Queries return correct results
- INSERT, UPDATE, DELETE modify the correct rows
