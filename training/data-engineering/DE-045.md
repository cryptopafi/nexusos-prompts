# DE-045 — Create Indexes for Query Optimization

## Problema
Missing or incorrect indexes cause full table scans, making queries orders of magnitude slower than necessary.

## Procedura
### Prerequisites
- Database with tables and queries to optimize

### Steps
1. Identify slow queries needing index optimization
2. Analyze query patterns to determine which columns to index
3. Create single-column indexes for frequently filtered columns
4. Create composite indexes for multi-column WHERE clauses
5. Create covering indexes that include all selected columns
6. Verify improvement with EXPLAIN before and after
7. Monitor index usage and remove unused indexes

### Verification
- Indexed queries show improved execution plans
- No unused or redundant indexes remain
