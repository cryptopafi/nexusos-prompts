# DE-039 — Schedule Jobs with Cron in Linux

## Problema
Data pipelines must run on schedule; cron provides reliable job scheduling for ETL and maintenance tasks.

## Procedura
### Prerequisites
- Linux terminal with cron
- Script to schedule

### Steps
1. Understand cron syntax: minute hour day month weekday
2. Edit crontab with crontab -e
3. Schedule a script to run daily, hourly, or weekly
4. Redirect output and errors to log files
5. List scheduled jobs with crontab -l
6. Test cron jobs by scheduling immediate execution
7. Implement monitoring to detect failed cron jobs

### Verification
- Cron job executes at the scheduled time
- Output and errors are logged correctly
