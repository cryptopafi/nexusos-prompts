# DE-048 — Implement ETL Pipeline Using Shell Scripts

## Problema
Shell-based ETL pipelines are lightweight and effective for simple data flows without requiring a framework.

## Procedura
### Prerequisites
- Shell scripting (DE-038)
- Source and destination systems

### Steps
1. Design the ETL flow: source > extract > transform > load
2. Write extraction scripts using curl, wget, or database CLI
3. Transform data using awk, sed, cut, or Python one-liners
4. Load data using database CLI bulk import
5. Chain ETL stages with shell script and error checking
6. Add logging for each stage
7. Schedule with cron for recurring execution

### Verification
- ETL pipeline runs end-to-end
- Data in destination matches expected transformations
