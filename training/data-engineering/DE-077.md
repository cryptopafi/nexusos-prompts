# DE-077 — Use GenAI for ETL Pipeline Code Generation

## Problema
Writing ETL pipeline boilerplate is tedious; GenAI can generate pipeline code from data flow descriptions.

## Procedura
### Prerequisites
- GenAI tool access
- ETL requirements documented

### Steps
1. Describe the ETL data flow in natural language
2. Specify the source format, transformations, and target
3. Ask the AI to generate Python ETL code
4. Review the generated code for correctness and best practices
5. Test the generated pipeline with sample data
6. Iterate with the AI to handle edge cases
7. Integrate the generated code into your pipeline framework

### Verification
- Generated ETL code runs without errors on sample data
- Code follows best practices (error handling, logging)
