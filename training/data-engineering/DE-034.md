# DE-034 — Navigate Linux File System with Shell Commands

## Problema
Data engineers work extensively in Linux environments; efficient navigation is foundational for all server-side tasks.

## Procedura
### Prerequisites
- Linux or macOS terminal access

### Steps
1. Use pwd to display current directory
2. Use ls with flags (-la, -lh) to list files
3. Use cd to navigate directories (absolute and relative paths)
4. Use find to search for files by name and type
5. Use tree to visualize directory structure
6. Understand the Linux directory hierarchy (/etc, /var, /home, /tmp)

### Verification
- Can navigate to any directory using both absolute and relative paths
- Can find files using find command
