# DE-063 — Set Up and Query Apache Cassandra

## Problema
Cassandra's distributed architecture handles massive write-heavy workloads that relational databases cannot scale.

## Procedura
### Prerequisites
- Docker or direct installation capabilities

### Steps
1. Install Apache Cassandra (Docker recommended for development)
2. Start Cassandra and connect with cqlsh
3. Create a keyspace with replication strategy
4. Create tables with appropriate partition and clustering keys
5. Insert data with INSERT statements
6. Query data with SELECT (respecting partition key requirements)
7. Understand Cassandra's consistency levels

### Verification
- Cassandra is running and accessible via cqlsh
- Data is written and read correctly
