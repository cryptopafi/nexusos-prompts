# DE-025 — Use COUNT, DISTINCT, and LIMIT in SQL

## Problema
Aggregate functions and result limiting are essential for data exploration and summary reporting.

## Procedura
### Prerequisites
- Database with sample data

### Steps
1. Use COUNT(*) and COUNT(column) to count rows
2. Use DISTINCT to find unique values
3. Combine COUNT with DISTINCT: COUNT(DISTINCT column)
4. Use LIMIT (or TOP) to restrict result size
5. Combine with ORDER BY for top-N queries
6. Use GROUP BY with COUNT for frequency distributions

### Verification
- Aggregate queries return correct counts
- DISTINCT eliminates duplicates correctly
