# DE-049 — Build Data Pipelines with Apache Airflow DAGs

## Problema
Complex data pipelines with dependencies, retries, and monitoring require a workflow orchestrator like Airflow.

## Procedura
### Prerequisites
- Python environment
- Understanding of DAG concepts

### Steps
1. Install Apache Airflow: pip install apache-airflow
2. Initialize the Airflow database: airflow db init
3. Define a DAG with default_args and schedule_interval
4. Create tasks using Operators (BashOperator, PythonOperator)
5. Define task dependencies with >> and <<
6. Test the DAG with airflow dags test
7. Monitor DAG runs in the Airflow web UI
8. Implement error handling with retries and alerts

### Verification
- DAG appears in Airflow UI
- Tasks execute in correct order with dependencies
