# DE-035 — Use Informational and File Management Linux Commands

## Problema
Managing files efficiently in Linux prevents data loss and keeps server environments organized.

## Procedura
### Prerequisites
- Linux terminal access

### Steps
1. Use cp, mv, rm for file operations
2. Use mkdir and rmdir for directory management
3. Use cat, head, tail, less for viewing file contents
4. Use wc for counting lines, words, characters
5. Use chmod and chown for permissions
6. Use du and df for disk usage monitoring

### Verification
- Can perform all file operations confidently
- Understand permission settings and disk usage
