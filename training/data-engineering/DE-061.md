# DE-061 — Set Up and Perform CRUD Operations in MongoDB

## Problema
MongoDB's document model is essential for handling semi-structured data that doesn't fit relational schemas.

## Procedura
### Prerequisites
- Admin access for installation

### Steps
1. Install MongoDB (brew install mongodb-community or apt install mongod)
2. Start the MongoDB service
3. Connect with mongosh CLI
4. Create a database and collection
5. Insert documents: insertOne() and insertMany()
6. Query documents: find() with filters and projections
7. Update documents: updateOne() and updateMany()
8. Delete documents: deleteOne() and deleteMany()

### Verification
- All CRUD operations execute successfully
- Documents are correctly stored and retrieved
