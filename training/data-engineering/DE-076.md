# DE-076 — Use GenAI for SQL Query Generation and Optimization

## Problema
Writing complex SQL queries is time-consuming; GenAI accelerates query development and suggests optimizations.

## Procedura
### Prerequisites
- Access to a GenAI tool (ChatGPT, Claude, Copilot)
- Database schema documentation

### Steps
1. Describe your data schema to the AI (table structures, relationships)
2. Ask the AI to generate a SQL query from a natural language question
3. Review the generated query for correctness
4. Ask the AI to optimize a slow query (provide the EXPLAIN output)
5. Use the AI to convert queries between SQL dialects
6. Validate AI-generated queries against expected results

### Verification
- AI-generated queries produce correct results
- Optimized queries show improved execution plans
