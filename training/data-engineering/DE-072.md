# DE-072 — Implement Feature Engineering Pipelines in Spark

## Problema
Feature engineering is repetitive and error-prone; Spark Pipelines standardize and automate the process.

## Procedura
### Prerequisites
- PySpark with MLlib
- Raw dataset requiring feature engineering

### Steps
1. Create individual transformers: StringIndexer, OneHotEncoder
2. Create a VectorAssembler to combine features
3. Build a Pipeline with stages in order
4. Fit the pipeline on training data
5. Transform both training and test data consistently
6. Save and load the pipeline model
7. Add custom transformers if needed

### Verification
- Pipeline transforms raw data into model-ready features
- Same pipeline transforms both train and test consistently
