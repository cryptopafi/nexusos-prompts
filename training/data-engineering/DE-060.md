# DE-060 — Create Dashboards and Reports with Looker Studio

## Problema
Multi-page reports and dashboards in Looker Studio enable comprehensive data storytelling for stakeholders.

## Procedura
### Prerequisites
- Looker Studio basics (DE-059)
- Data source connected

### Steps
1. Create a multi-page report with navigation
2. Add summary KPIs on the first page
3. Create detail pages with drill-down charts
4. Configure page-level and report-level filters
5. Add calculated fields for custom metrics
6. Share the report with view and edit permissions
7. Schedule email delivery of the report

### Verification
- Multi-page report navigates correctly
- Calculated fields produce accurate results
