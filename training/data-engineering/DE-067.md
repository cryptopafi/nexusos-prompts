# DE-067 — Process Data with Spark DataFrames and Spark SQL

## Problema
Spark DataFrames provide an optimized, SQL-friendly interface for big data processing with automatic query optimization.

## Procedura
### Prerequisites
- PySpark installed (DE-066)
- Sample datasets

### Steps
1. Create a SparkSession
2. Load data into DataFrames from CSV, JSON, Parquet
3. Perform DataFrame operations: select, filter, groupBy, agg
4. Register a DataFrame as a temporary view
5. Write Spark SQL queries against the view
6. Join multiple DataFrames
7. Write results to Parquet or other output formats

### Verification
- DataFrame operations and SQL queries produce identical results
- Output is correctly written to the target format
