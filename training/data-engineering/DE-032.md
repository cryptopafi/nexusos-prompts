# DE-032 — Access Databases from Python Using DB-API

## Problema
Python DB-API provides a standard interface for database access, enabling programmatic data pipeline development.

## Procedura
### Prerequisites
- Python installed
- Database accessible

### Steps
1. Install a database driver (mysql-connector-python or psycopg2)
2. Establish a database connection with connect()
3. Create a cursor object
4. Execute SQL queries with cursor.execute()
5. Fetch results with fetchone(), fetchall(), fetchmany()
6. Use parameterized queries to prevent SQL injection
7. Close cursor and connection properly (use context managers)

### Verification
- Python can connect, query, and retrieve data from the database
- Parameterized queries prevent SQL injection
