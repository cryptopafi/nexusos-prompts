# DE-017 — Create Tables with DDL

## Problema
Data engineers must write DDL statements to create and modify database schemas reliably and consistently.

## Procedura
### Prerequisites
- Database access (MySQL or PostgreSQL)

### Steps
1. Write CREATE TABLE statements with appropriate data types
2. Define NOT NULL and DEFAULT constraints
3. Add CHECK constraints for data validation
4. Create tables with AUTO_INCREMENT/SERIAL primary keys
5. Write ALTER TABLE statements to modify existing tables
6. Write DROP TABLE with IF EXISTS for safety
7. Version control your DDL scripts

### Verification
- Tables are created with all specified constraints
- DDL scripts are idempotent (can be run multiple times safely)
