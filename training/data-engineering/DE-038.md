# DE-038 — Write Bash Shell Scripts

## Problema
Automating repetitive data engineering tasks with shell scripts saves time and reduces human error.

## Procedura
### Prerequisites
- Linux terminal with bash

### Steps
1. Create a script file with #!/bin/bash shebang
2. Use variables and command substitution
3. Implement conditional logic with if/elif/else
4. Use for and while loops for iteration
5. Handle command-line arguments with $1, $2, $@
6. Implement error handling with set -e and trap
7. Make scripts executable with chmod +x

### Verification
- Script runs without errors
- Script handles edge cases (missing args, empty files)
