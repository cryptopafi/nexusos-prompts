# DE-021 — Create Databases, Tables, and Load Data in MySQL

## Problema
Loading data into MySQL requires understanding schema creation, data types, and bulk import techniques.

## Procedura
### Prerequisites
- MySQL installed and running (DE-020)
- Data files to load

### Steps
1. Create a new database with CREATE DATABASE
2. Design and create tables with appropriate data types
3. Load data from CSV using LOAD DATA INFILE
4. Alternatively use mysqlimport for bulk loading
5. Verify data integrity after loading
6. Create indexes on frequently queried columns

### Verification
- Data is loaded completely and matches source
- Indexes are created and query performance is acceptable
