# DE-050 — Build Streaming Data Pipelines with Apache Kafka

## Problema
Batch processing is insufficient for real-time data needs; Kafka enables streaming pipelines for low-latency data flows.

## Procedura
### Prerequisites
- Java runtime installed
- Understanding of pub/sub messaging

### Steps
1. Install and start Apache Kafka and Zookeeper
2. Create a Kafka topic for your data stream
3. Write a producer to publish messages to the topic
4. Write a consumer to read and process messages
5. Implement message serialization/deserialization (JSON, Avro)
6. Handle consumer group coordination for parallel processing
7. Monitor Kafka with built-in tools or Kafka Manager

### Verification
- Messages flow from producer to consumer without loss
- Consumer groups process messages in parallel correctly
