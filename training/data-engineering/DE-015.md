# DE-015 — Package Python Applications with Unit Testing

## Problema
Production data pipelines need proper packaging and testing to ensure reliability during deployments.

## Procedura
### Prerequisites
- Python project to package
- pytest installed

### Steps
1. Structure project with src/ and tests/ directories
2. Create setup.py or pyproject.toml for packaging
3. Write unit tests for core functions using pytest
4. Implement test fixtures for data setup/teardown
5. Add integration tests for database or API interactions
6. Run tests with coverage reporting
7. Package the application for deployment

### Verification
- All unit tests pass
- Coverage report shows >80% for core modules
- Package installs correctly
