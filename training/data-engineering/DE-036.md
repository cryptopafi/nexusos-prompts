# DE-036 — Archive and Compress Files in Linux

## Problema
Data files are large; archiving and compression saves storage and speeds up data transfers.

## Procedura
### Prerequisites
- Linux terminal with tar, gzip, zip installed

### Steps
1. Create a tar archive: tar -cvf archive.tar files/
2. Create a compressed tar.gz: tar -czvf archive.tar.gz files/
3. Extract a tar archive: tar -xvf archive.tar
4. Extract a tar.gz: tar -xzvf archive.tar.gz
5. Use zip and unzip for cross-platform compatibility
6. Use gzip and gunzip for single file compression

### Verification
- Archives are created and contain all expected files
- Extracted files match originals
