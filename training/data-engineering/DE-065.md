# DE-065 — Set Up Hadoop Ecosystem

## Problema
Hadoop provides the foundation for big data processing but its ecosystem setup is complex without guidance.

## Procedura
### Prerequisites
- Java 8+ installed
- Linux or macOS environment

### Steps
1. Install Hadoop in pseudo-distributed mode (or use Docker)
2. Configure core-site.xml and hdfs-site.xml
3. Format the HDFS namenode
4. Start HDFS and YARN services
5. Verify HDFS is working: hadoop fs -ls /
6. Upload a test file to HDFS
7. Access the HDFS and YARN web UIs

### Verification
- HDFS and YARN services are running
- Files can be uploaded and retrieved from HDFS
