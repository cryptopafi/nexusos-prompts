# DE-059 — Build Data Visualizations with Google Looker Studio

## Problema
Google Looker Studio provides free, powerful visualization connected to BigQuery and other Google data sources.

## Procedura
### Prerequisites
- Google account
- Data source accessible

### Steps
1. Open Looker Studio and create a new report
2. Connect to a data source (BigQuery, Sheets, CSV)
3. Add charts: time series, bar, pie, geo map, scorecard
4. Configure metrics and dimensions for each chart
5. Apply date range controls and filters
6. Style the report with consistent branding

### Verification
- Visualizations display correct data
- Report is styled and professional
