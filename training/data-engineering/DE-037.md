# DE-037 — Manage Linux Networking Commands

## Problema
Data engineers troubleshoot network issues affecting data pipelines and configure secure data transfers.

## Procedura
### Prerequisites
- Linux terminal with network access

### Steps
1. Use ping to test connectivity
2. Use curl and wget for HTTP requests and downloads
3. Use ssh for secure remote access
4. Use scp and rsync for secure file transfers
5. Use netstat or ss to check open ports
6. Use hostname and ifconfig/ip for network info

### Verification
- Can diagnose basic network connectivity issues
- Can transfer files securely between machines
