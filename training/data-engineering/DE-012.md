# DE-012 — Web Scrape Data Using BeautifulSoup

## Problema
Some data sources lack APIs; web scraping extracts structured data from HTML pages for analysis.

## Procedura
### Prerequisites
- Python with requests and beautifulsoup4
- Target website identified

### Steps
1. Install BeautifulSoup: pip install beautifulsoup4
2. Fetch a web page with requests
3. Parse HTML with BeautifulSoup
4. Navigate the DOM tree to find target elements
5. Extract text and attributes from elements
6. Handle pagination across multiple pages
7. Save scraped data to CSV or JSON
8. Respect robots.txt and rate limits

### Verification
- Scraped data matches what's visible on the web page
- Script respects rate limits and robots.txt
