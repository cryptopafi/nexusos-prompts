# DE-026 — Filter SQL Results with String Patterns and Ranges

## Problema
Complex filtering with LIKE, BETWEEN, and IN enables precise data selection for analytics and reporting.

## Procedura
### Prerequisites
- Database with varied sample data

### Steps
1. Use LIKE with % and _ wildcards for pattern matching
2. Use NOT LIKE to exclude patterns
3. Use BETWEEN for range filtering (dates, numbers)
4. Use IN for matching against a list of values
5. Use NOT IN to exclude specific values
6. Combine multiple filters with AND and OR
7. Use parentheses for complex filter logic

### Verification
- Filters return exactly the expected rows
- Complex combinations of filters work correctly
