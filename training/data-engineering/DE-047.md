# DE-047 — Automate Database Tasks with Scripts

## Problema
Manual database maintenance is error-prone and inconsistent; automation ensures tasks run correctly every time.

## Procedura
### Prerequisites
- Database access
- Shell scripting skills (DE-038)

### Steps
1. Identify repetitive database tasks (backups, cleanups, reports)
2. Write a shell script for each task
3. Use database CLI tools (psql, mysql) within scripts
4. Add error handling and logging to each script
5. Schedule scripts with cron
6. Set up notifications for task success/failure
7. Test scripts in a non-production environment first

### Verification
- Automated tasks run on schedule without errors
- Logs capture execution details
