# DE-033 — Use SQL Magic Commands in Jupyter Notebooks

## Problema
SQL magic commands in Jupyter enable interactive SQL exploration alongside Python, ideal for data analysis workflows.

## Procedura
### Prerequisites
- Jupyter Notebook (DE-006)
- Database accessible

### Steps
1. Install ipython-sql: pip install ipython-sql
2. Load the extension: %load_ext sql
3. Connect to database: %sql connection_string
4. Run single-line queries: %sql SELECT ...
5. Run multi-line queries: %%sql
6. Store query results in Python variables
7. Combine SQL queries with pandas for further analysis

### Verification
- SQL queries execute within notebook cells
- Results are accessible as Python objects
