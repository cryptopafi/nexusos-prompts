# DE-046 — Troubleshoot Common Database Issues

## Problema
Database issues (locks, deadlocks, connection exhaustion) require systematic diagnosis to resolve quickly.

## Procedura
### Prerequisites
- Database admin access
- Access to logs and monitoring

### Steps
1. Identify symptoms: slow queries, timeouts, connection errors
2. Check database logs for error messages
3. Monitor active connections and running queries
4. Diagnose lock contention and deadlocks
5. Identify and kill long-running queries if needed
6. Check disk space and storage health
7. Document the issue, diagnosis, and resolution

### Verification
- Issue is diagnosed and root cause identified
- Resolution is documented for future reference
