# DE-078 — Leverage GenAI for Data Quality Checks and Validation

## Problema
Manual data quality checks miss issues; GenAI helps generate comprehensive validation rules from data descriptions.

## Procedura
### Prerequisites
- GenAI tool access
- Dataset to validate

### Steps
1. Describe your dataset and expected quality standards
2. Ask the AI to generate data validation rules
3. Review rules for completeness (nulls, types, ranges, patterns)
4. Ask the AI to generate Python validation code (Great Expectations or custom)
5. Run validation checks on your dataset
6. Review the validation report
7. Iterate to add business-specific quality rules

### Verification
- Validation rules catch known data quality issues
- Validation report is comprehensive and actionable
