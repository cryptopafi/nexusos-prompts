# DE-062 — Write MongoDB Aggregation Pipelines

## Problema
Simple find queries are insufficient for complex analytics; aggregation pipelines provide SQL-like analytical capabilities.

## Procedura
### Prerequisites
- MongoDB with data (DE-061)

### Steps
1. Understand the aggregation pipeline concept (stages in sequence)
2. Use $match to filter documents
3. Use $group to aggregate (sum, avg, count)
4. Use $sort to order results
5. Use $project to reshape documents
6. Use $lookup for joins between collections
7. Chain multiple stages for complex transformations

### Verification
- Aggregation pipeline produces correct results
- Can chain 3+ stages for complex analytics
