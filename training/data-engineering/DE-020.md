# DE-020 — Set Up and Configure MySQL Database

## Problema
MySQL is one of the most widely used databases; proper setup and configuration ensures reliable operation.

## Procedura
### Prerequisites
- Admin access to the machine

### Steps
1. Install MySQL Server (brew install mysql or apt install mysql-server)
2. Secure the installation with mysql_secure_installation
3. Create a database user with appropriate privileges
4. Configure key settings: innodb_buffer_pool_size, max_connections
5. Test connection with mysql CLI client
6. Create a sample database and table

### Verification
- MySQL server is running and accessible
- User can connect and create databases
