# DE-011 — Access REST APIs and Collect Data with Python

## Problema
Many data sources are only accessible via APIs; engineers must know how to authenticate, paginate, and handle errors.

## Procedura
### Prerequisites
- Python installed
- API endpoint to access

### Steps
1. Install the requests library: pip install requests
2. Make a GET request to a public API
3. Parse JSON response data
4. Handle authentication (API keys, OAuth tokens)
5. Implement pagination to collect all data
6. Handle rate limiting with retries and backoff
7. Save collected data to a file or database

### Verification
- API data is collected completely (all pages)
- Error handling manages rate limits gracefully
