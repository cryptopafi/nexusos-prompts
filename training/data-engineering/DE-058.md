# DE-058 — Create Interactive Dashboards in IBM Cognos

## Problema
Static reports require manual refresh; interactive dashboards enable self-service exploration by business users.

## Procedura
### Prerequisites
- IBM Cognos with data visualizations (DE-057)

### Steps
1. Create a new dashboard in Cognos
2. Add multiple visualizations to the dashboard canvas
3. Configure cross-filtering between visualizations
4. Add global filters and date range selectors
5. Implement drill-down navigation for detailed views
6. Set up auto-refresh for real-time data
7. Share the dashboard with appropriate access controls

### Verification
- Dashboard updates when filters are applied
- Drill-down navigation works correctly
