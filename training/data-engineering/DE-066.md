# DE-066 — Configure and Use Apache Spark RDDs

## Problema
RDDs are Spark's foundational abstraction for distributed data processing and understanding them is key to Spark mastery.

## Procedura
### Prerequisites
- PySpark installed
- Sample data files

### Steps
1. Install PySpark: pip install pyspark
2. Create a SparkContext
3. Create RDDs from collections and files
4. Apply transformations: map, filter, flatMap, reduceByKey
5. Execute actions: collect, count, take, saveAsTextFile
6. Understand lazy evaluation and the DAG execution model
7. Monitor Spark jobs in the Spark UI

### Verification
- RDD operations produce correct results
- Understand the difference between transformations and actions
