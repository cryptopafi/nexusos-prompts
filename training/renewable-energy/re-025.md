# RE-025 — Plan Solar Technology for Project

## Problema
Technology selection for a specific solar project requires balancing performance, cost, bankability, and site constraints.

## Procedura
### Prerequisites
- RE-007 completed
- RE-021 completed

### Steps
1. Evaluate PV module technology options against project requirements (mono-PERC, TOPCon, HJT, thin-film)
2. Select mounting system: fixed tilt, single-axis tracker, dual-axis tracker based on site latitude and economics
3. Specify inverter architecture: central, string, or microinverter based on project scale
4. Design DC collection system and string sizing
5. Evaluate bifacial module suitability based on ground albedo and row spacing
6. Assess module bankability and manufacturer financial health
7. Finalize technology package with performance modeling validation

### Verification
- Technology selection justified with quantitative comparison (yield, cost, LCOE)
- Selected modules from Tier 1 bankable manufacturer list
