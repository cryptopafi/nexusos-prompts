# RE-007 — Design Solar PV System (Cell to Power Plant)

## Problema
Understanding the full PV system hierarchy from cell to utility-scale plant is essential for proper design and optimization.

## Procedura
### Prerequisites
- RE-006 completed
- Basic electrical engineering knowledge

### Steps
1. Select PV cell technology (mono-Si, poly-Si, thin-film) based on efficiency and cost targets
2. Design module configuration (cells in series/parallel for target voltage and current)
3. Size string inverter or microinverter based on module specifications
4. Design array layout considering row spacing and ground coverage ratio (GCR)
5. Specify balance of system (BOS): wiring, combiner boxes, transformers
6. Calculate system losses (soiling, mismatch, wiring, inverter clipping)
7. Estimate system yield (kWh/kWp) and performance ratio

### Verification
- Performance ratio between 75-85% for well-designed system
- DC/AC ratio within 1.1-1.3 range
