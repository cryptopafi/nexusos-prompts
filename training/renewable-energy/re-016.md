# RE-016 — Forecast Renewable Energy Production for Grid Management

## Problema
Inaccurate renewable generation forecasts cause grid imbalances, increased reserve costs, and potential reliability issues.

## Procedura
### Prerequisites
- RE-003 or RE-006 completed
- RE-014 completed

### Steps
1. Select forecast horizon: day-ahead, intra-day, hour-ahead, or real-time
2. Identify weather data sources (NWP models, satellite, ground stations)
3. Choose forecasting methodology: physical models, statistical models, or machine learning
4. Implement ensemble forecasting to quantify uncertainty
5. Calculate forecast error metrics: MAE, RMSE, skill score
6. Design ramp event detection and alerting system
7. Integrate forecasts into grid operator scheduling workflow

### Verification
- Forecast skill score positive compared to persistence model
- Ramp event detection captures >80% of significant ramps
