# RE-029 — Evaluate Renewable Energy Project Financial Metrics

## Problema
Investors and lenders evaluate RE projects using specific financial metrics that differ from standard corporate finance.

## Procedura
### Prerequisites
- RE-028 completed
- RE-009 completed

### Steps
1. Calculate project NPV using after-tax cash flows
2. Determine project IRR (unlevered) and equity IRR (levered)
3. Calculate Debt Service Coverage Ratio (DSCR) for each year of debt term
4. Compute payback period (simple and discounted)
5. Analyze cash-on-cash return by year
6. Evaluate minimum DSCR vs average DSCR against lender requirements (typically >1.3x)
7. Perform scenario analysis: base case, downside, upside

### Verification
- Financial metrics calculated for all three scenarios
- DSCR profile meets lender minimum requirements in downside case
