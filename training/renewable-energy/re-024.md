# RE-024 — Conduct Stakeholder Analysis for RE Project

## Problema
Projects that ignore stakeholder concerns face opposition, delays, and sometimes cancellation regardless of technical merit.

## Procedura
### Prerequisites
- RE-021 completed

### Steps
1. Identify all stakeholder groups: landowners, community, government, utilities, environmental groups, tribal nations
2. Map stakeholder interests, concerns, and influence levels
3. Classify stakeholders using power-interest grid
4. Develop engagement strategy for each stakeholder category
5. Design community benefit program (jobs, tax revenue, community solar, landowner payments)
6. Plan public meetings and comment response process
7. Establish ongoing communication channels throughout project lifecycle

### Verification
- All stakeholder groups identified and mapped
- Engagement strategy addresses top 3 concerns for each high-influence stakeholder
