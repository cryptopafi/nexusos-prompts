# RE-026 — Plan Wind Technology for Project

## Problema
Wind turbine selection directly determines project revenue and must match site conditions, grid requirements, and logistics constraints.

## Procedura
### Prerequisites
- RE-005 completed
- RE-021 completed

### Steps
1. Define turbine selection criteria: IEC class, rotor diameter, hub height, specific power
2. Shortlist 3-5 turbine models from major OEMs matching site wind class
3. Run energy yield analysis for each candidate turbine using site wind data
4. Evaluate noise emissions against local regulations and setback requirements
5. Assess turbine transportation and installation logistics for site access
6. Compare warranty terms, service agreements, and OEM financial stability
7. Select optimal turbine based on AEP/cost ratio and bankability

### Verification
- Energy yield analysis completed for all shortlisted turbines
- Selected turbine IEC class matches or exceeds site wind conditions
